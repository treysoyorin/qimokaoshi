## 绪论（2学时）

### 课件定位
老师开篇就说：化工原理是**"从基础课程到专业课程的桥梁"**，目的是**"初步建立起工程观念"**。不是纯理论课，是面向实际生产问题的技术基础课。

### 核心知识点

1. **单元操作（Unit Operations）**
   - 定义：化工生产过程中共有的物理操作，只改变物料状态/物理性质，不改变化学性质
   - 三过程：原料预处理 → 化学反应 → 产物分离提纯
   - 三大类：
     - 流体动力过程：流体输送、沉降、过滤、固体流态化
     - 热过程：传热、冷凝、蒸发
     - 传质过程：蒸馏、吸收、萃取、结晶、干燥
   - 特点：不同化工生产中单元操作数目/名称/顺序不同，但**基本原理和设备通用**

2. **化工原理的任务**
   - 选型：根据工艺要求选择单元操作和设备
   - 设计：设备设计和工艺计算
   - 操作：操作原理、操作方法、故障分析

3. **三种衡算（基本分析方法）**
   - 质量衡算：输入 = 输出 + 积累（稳态时积累=0）
   - 能量衡算：输入能量 = 输出能量 + 损失
   - 动量衡算：过程速率 = 推动力 / 阻力

4. **研究方法**
   - 理论模型：精确但难做到
   - 经验模型：回归数据，无外推性
   - **半经验半理论模型**：理论推导表达式 + 实验确定参数（★重点！）
   - 人工智能模型

5. **单位换算（必须非常熟悉）**
   - 1 atm = 1.033 kgf/cm² = 760 mmHg = 10.33 mH₂O = 1.0133 bar = 101325 Pa
   - 1 kgf/cm² = 1 at = 9.80×10⁴ Pa = 735.6 mmHg = 10 mH₂O
   - 1 kgf = 9.81 N

6. **量纲分析**
   - 依据：量纲一致性原则 + π定理
   - 无量纲数群个数 = 变量数 - 基本量纲个数
   - 应用：湍流摩擦系数、对流传热系数

### 重点例题（绪论）

**例0-1：硝酸钾生产物料衡算（带循环）**
- 20% KNO₃溶液1000kg/h → 蒸发至50% → 结晶得含水4%结晶 + 37.5%饱和溶液（循环）
- 求：结晶产品量P、水蒸发量W、循环量R
- **步骤**：
  1. 对整个系统（Ⅰ）做KNO₃衡算：1000×0.2 = 0.96P → P = 208.3 kg/h
  2. 对整个系统做总物料衡算：1000 = P + W → W = 791.7 kg/h
  3. 对冷却结晶器（Ⅱ）做衡算：S = P + R，S×0.5 = P×0.96 + R×0.375 → R = 766.6 kg/h
- **关键**：选系统边界（整个流程 vs 单个设备），稳态时积累=0

**例0-2：换热器热量衡算（热损失百分数）**
- 溶液：Cp=3.56kJ/(kg·°C)，1.0kg/s，25°C→80°C
- 蒸汽：120°C饱和蒸汽，0.095kg/s，冷凝成120°C饱和水
- 求：热损失占蒸汽提供热量的百分数
- **步骤**：
  1. 查120°C饱和蒸汽焓=2708.9kJ/kg，饱和水焓=503.67kJ/kg
  2. 输入热量 = 蒸汽带入 + 溶液带入 = 0.095×2708.9 + 1.0×3.56×25 = 257.3 + 89 = 346.3 kW
  3. 输出热量 = 冷凝水带出 + 溶液带出 = 0.095×503.67 + 1.0×3.56×80 = 47.8 + 284.8 = 332.6 kW
  4. 热损失 = 346.3 - 332.6 = 13.7 kW
  5. 损失百分数 = 13.7 / (257.3 - 47.8) = 6.54%
- **关键**：蒸汽放热是冷凝潜热（r = 2708.9 - 503.67），不是CpΔT！

### 常见陷阱
- 物料衡算中把"质量分数"和"摩尔分数"混用
- 带循环题中，循环量和总进料量混淆（注意区分总进料和新鲜进料）
- 能量衡算中忽略相变潜热（蒸汽冷凝时放热是r，不是Cp·ΔT）
- 单位不统一：质量用kg，体积用m³，密度用kg/m³

## 第一章 1.1 流体基本概念

### 核心知识点

1. **流体定义**：气体（含蒸汽）和液体总称为流体
   - 特征：流动性、无固定形状、外力作用下内部发生相对运动

2. **连续性假设**：视流体为由大量质点组成的连续介质
   - 质点大小：相对于管道无限小，相对于分子无限大

3. **密度**
   - **理想气体**：ρ = PM/(RT)（公式3）
     - 标准状态→操作状态：ρ = ρ₀(P/P₀)(T₀/T)
   - **理想气体混合物**：ρ_m = Σ(y_i·M_i)·P/(RT) = PM_m/(RT)（公式4、5、6、7）
     - 注意：y_i是摩尔分数（=体积分率=压强分率）
   - **液体混合物**：1/ρ_m = Σ(w_i/ρ_i)（公式8）
     - 注意：w_i是质量分数，不是算术平均！

4. **例1-1：混合气体密度计算**
   - HCl混合气体：25% HCl，75%空气（体积分数）
   - T=50°C=323K，P=743mmHg
   - M_HCl=36.5，M_空气=29
   - M_m = 0.25×36.5 + 0.75×29 = 30.875 kg/kmol
   - ρ = PM_m/(RT) = (743/760)×101325×30.875/(8314×323)
   - 或用公式：ρ_m = Σ(y_i·M_i)·P/(RT)

5. **例1-2：混合液体密度计算**
   - 60%（质量分数）醋酸水溶液，293K
   - ρ_水=998，ρ_醋酸=1049
   - 1/ρ_m = 0.6/1049 + 0.4/998 → ρ_m = 1019 kg/m³

### 常见陷阱
- 气体密度计算时状态条件不统一（标准状态 vs 操作状态）
- 混合液体密度用算术平均：ρ_m ≠ 0.6×1049 + 0.4×998（错误！）
- 单位：M用kg/kmol，R=8314 J/(kmol·K)，P用Pa，T用K

## 第一章 1.2 流体静力学（★★★重点）

### 核心知识点

1. **压强定义**：流体单位面积上承受的垂直作用力，Pa = N/m²

2. **压强表示方法**
   - 绝对压强：以绝对零压为起点
   - 表压 = 绝对压强 - 大气压
   - 真空度 = 大气压 - 绝对压强
   - 真空度称为负压
   - **注意**：大气压必须指当时当地的大气压

3. **静力学基本方程**
   - 推导：微元体受力分析，静力平衡
   - 方程：P₂ = P₁ + ρgh（公式8、9、10）
   - 或：P/ρ + gz = 常数
   - 适用条件：重力场中静止、连续、同一种流体、密度为常数

4. **物理意义**
   - 等压面：静止、连续、均一、水平面上各点压强相同
   - **传递定律**：液面上方压强改变时，液体内部各点压强以同样大小变化
   - 液柱高度表示压强：P/ρg = h（必须注明是何种液体）

5. **U形管压差计**
   - 关键：正确确定等压面
   - 公式推导：
     - 取等压面a-a'，P_a = P_a'
     - P_A + ρ_B·g·Z = P_a
     - P_a' = P_B + ρ_B·g·(Z+R) + ρ_A·g·R
     - 联立得：ΔP = P_A - P_B = (ρ_A - ρ_B)gR + ρ_B·g·Z
     - 当水平管Z=0时：ΔP = (ρ_A - ρ_B)gR
   - **测量气体时**：ρ_B << ρ_A，简化为 ΔP ≈ ρ_A·g·R
   - **特点**：ΔP仅与R和密度差有关，与U形管粗细/长短/位置无关

6. **液面测量**
   - 液柱压差计：液面高度与压差计读数的关系
   - 鼓泡式液柱测量：压缩氮气经调节阀进入，鼓泡观察器控制流速，U形管计量压差

7. **液封**
   - 作用：防止气体泄漏、设备安全（超压时排气）
   - 液封高度：h = P/(ρg)（P为设备内外压差）

### 重点例题

**例1-3：真空度计算（海拔影响！）**
- 天津：大气压101.33kPa，兰州：85.3kPa
- 塔顶绝对压强=5.3kPa
- 天津真空度 = 101.33 - 5.3 = 96.03 kPa
- 兰州真空度 = 85.3 - 5.3 = 80 kPa
- **关键**：海拔越高，大气压越低，相同绝对压强对应的真空度越小

**例1-4：等压面判断（★经典题）**
- 开口容器：油层h₁=0.8m，ρ₁=800；水层h₂=0.6m，ρ₂=1000
- 判断P_A = P_A'是否成立？P_B = P_B'是否成立？
- **分析**：
  - A-A'：在静止连通同一种流体内且同一水平面 → **是等压面** ✓
  - B-B'：在静止流体同一水平面，但**不是连通同一种流体**（油和水） → **不是等压面** ✗
- 计算水在玻璃管内高度h：
  - P_A = P_A' → P_0 + ρ₁gh₁ + ρ₂gh₂ = P_0 + ρ₂gh
  - h = (ρ₁h₁ + ρ₂h₂)/ρ₂ = (800×0.8 + 1000×0.6)/1000 = 1.24 m

**例1-5：复式U形管压差计（★考试常客）**
- 管内液体密度ρ=900，指示剂水银ρ_A=13600
- R₁=0.35m，R₂=0.45m
- 求A、B两点压差
- **步骤**：取等压面1-1'、2-2'、3-3'、4-4'，逐段列静力学方程
  - P_A + ρg(z_A - z₁) = P₁
  - P₁ = P₁' + ρ_A·g·R₁
  - P₂ = P₂' - ρ·g·(z₂ - z₂')
  - ...（中间段液体为水银）
  - 最终：ΔP = P_A - P_B = (ρ_A - ρ)g(R₁ + R₂)
  - = (13600 - 900)×9.81×0.8 = 99.7 kPa
- **关键**：取多个等压面，逐段列方程，最后相加

**例1-6：液面测量（鼓泡式）**
- U形管水银读数R=130mm=0.13m
- 有机液体密度ρ=1250
- 储罐上方与大气相通
- 求液面离吹气管出口距离h
- **分析**：管出口a处压力≈U形管b处压力
  - P_a = P_b → ρ_有机·g·h = ρ_水银·g·R
  - h = ρ_水银·R/ρ_有机 = 13600×0.13/1250 = 1.414 m

**例1-7：液封高度计算**
- 真空表读数=86×10³ Pa（即P_绝 = P_大气 - 86000）
- 气压管中水上升高度h
- 分析：P_大气 = P_绝 + ρgh → h = (P_大气 - P_绝)/(ρg) = 86000/(1000×9.81) = 8.77 m

### 常见陷阱
- 等压面条件漏一个：必须是静止、连通、同种、水平四个条件
- 不同流体交界面上假设压强相等：油水界面不是等压面
- U形管计算时忽略被测流体密度：ΔP = (ρ_指 - ρ_被测)gR，不是ρ_指·g·R
- 液柱高度表示压强时，不注明液体种类：10mH₂O ≠ 10mHg

## 第一章 1.3 流体流动基本概念

### 核心知识点

1. **流量与流速**
   - 体积流量V_s（m³/s），质量流量m_s（kg/s）
   - 平均流速u = V_s/A（m/s）
   - 质量流速G = m_s/A = ρu（kg/m²·s）
   - 常用流速范围：水1~3m/s，低压气体8~15m/s

2. **稳态 vs 非稳态**
   - 稳态：物理量仅随位置变化，不随时间变化 u=f(x,y,z)
   - 非稳态：物理量既随位置又随时间变化 u=f(x,y,z,θ)
   - 连续化生产正常情况下多为稳态

3. **牛顿黏性定律**
   - τ = μ(du/dy)
   - 剪应力与速度梯度成正比
   - 动力粘度μ：Pa·s；1cP = 10⁻³Pa·s = 1mPa·s
   - 运动粘度ν = μ/ρ（m²/s）
   - 液体粘度：随温度升高而降低，压力影响很小
   - 气体粘度：随温度升高而增大，高压时随压力增加

4. **雷诺实验与流动型态**
   - **层流**：Re≤2000，质点分层流动，一维流动，互不混合
   - **湍流**：Re≥4000，质点沿轴向流动但有径向脉动，位置交换，剧烈碰撞
   - **过渡区**：2000<Re<4000，不稳定
   - **本质区别**：流体内部质点的运动方式不同

5. **雷诺数**
   - Re = ρud/μ = 惯性力/黏性力
   - 物理意义：反映惯性力与黏性力的对比，标志湍动程度

6. **速度分布**
   - **层流**：抛物线分布，u = u_max[1-(r/R)²]，u_avg = u_max/2
   - **湍流**：较平坦，u = u_max(1-r/R)^(1/n)，n≈7时u_avg≈0.82u_max

7. **边界层**
   - 定义：流速降为主体流速99%以内的区域
   - 层流内层：靠近壁面，速度梯度大，以分子黏度为主
   - 湍流主体：速度脉动大，以湍流黏度为主
   - **层流内层是传递过程的主要阻力**
   - Re越大，层流内层越薄

## 第一章 1.4 伯努利方程（★★★★★核心）

### 核心知识点

1. **推导思路**（课件明确给出）
   - 总能量衡算（含内能、热能）→ 机械能衡算 → 不可压缩流体稳态流动机械能衡算（伯努利方程）

2. **流体具有的能量**
   - 内能U（J/kg）：状态函数，与温度有关
   - 位能：gz（J/kg）
   - 动能：u²/2（J/kg）
   - 静压能：P/ρ（J/kg）——将流体推进流动体系所需的功
   - 总能量：U + gz + u²/2 + P/ρ

3. **系统与外界交换的能量**
   - 热Q_e（J/kg）：吸热为正
   - 功W_e（J/kg）：外界对流体做功

4. **总能量衡算式**
   - U₁ + gz₁ + u₁²/2 + P₁/ρ + Q_e + W_e = U₂ + gz₂ + u₂²/2 + P₂/ρ
   - 稳态：输入能量 = 输出能量

5. **伯努利方程推导**
   - 由热力学第一定律：ΔU = Q_e' - PΔV = Q_e + Σh_f - PΔV
   - 整理得机械能衡算式：
     ```
     gz₁ + u₁²/2 + P₁/ρ + W_e = gz₂ + u₂²/2 + P₂/ρ + Σh_f
     ```
   - 对不可压缩流体（ρ=常数），简化为：
     ```
     gz₁ + u₁²/2 + P₁/ρ + W_e = gz₂ + u₂²/2 + P₂/ρ + Σh_f
     ```

6. **伯努利方程讨论**
   - **理想流体**：μ=0，Σh_f=0，无外加功W_e=0
     - gz + u²/2 + P/ρ = 常数
     - 总机械能守恒，各项可相互转化
   - **压头形式**（除以g）：
     ```
     z + u²/(2g) + P/(ρg) + H_e = z + u²/(2g) + P/(ρg) + H_f
     ```
     - 每一项单位：m（液柱高度）
   - **有外加功**：H_e = W_e/g，表示泵提供的压头
   - **有能量损失**：H_f = Σh_f/g，表示阻力损失压头

7. **应用注意事项**
   - 适用条件：不可压缩、稳态、连续
   - 截面选取：两截面间流体连续，中间无泵或换热器时W_e=0
   - 基准面：z是相对高度，选定后不能换
   - 单位统一：所有项单位一致（J/kg或m）

### 常见陷阱
- 压强项单位不一致：P₁用Pa，P₂用mmHg，必须统一
- 基准面乱换：z₁和z₂必须相对于同一基准面
- 动压头漏掉：u²/(2g)在高速流动时不可忽略
- 泵在方程中的位置：W_e是外界对单位质量流体做的功
- 理想流体假设：实际流体μ≠0，Σh_f≠0

## 第一章 1.5 流体流动阻力计算（★★★★★期末大题核心）

### 核心知识点

1. **阻力产生根源**
   - 流体具有黏性 → 流动时存在内部摩擦力
   - 固定管壁 → 产生速度梯度
   - 阻力损失Σh_f与流体内部结构密切相关

2. **阻力分类**
   - 直管阻力：流体流经一定管径直管时，由于内摩擦力产生的阻力
   - 局部阻力：流经管件、阀门、流量计等局部障碍引起的阻力

3. **能量损失表示**
   - Σh_f：单位质量流体损失的机械能（J/kg）
   - ΣH_f：单位重量流体损失的机械能（m液柱）
   - ΔP_f：单位体积流体损失的机械能（Pa）
   - 关系：ΣH_f = Σh_f/g，ΔP_f = ρ·Σh_f

4. **范宁公式（★核心公式）**
   ```
   h_f = λ(l/d)(u²/2g)        [压头损失]
   ΔP_f = λ(l/d)(ρu²/2)      [压强损失]
   ```
   - λ：摩擦系数，无量纲
   - 适用：层流与湍流，管道水平/垂直/倾斜

5. **层流摩擦系数**
   - 哈根-泊谡叶公式推导：ΔP_f = 32μlu/d²
   - 与范宁公式对比：λ = 64/Re
   - **精确公式，可直接推导**

6. **湍流摩擦系数**
   - 影响因素：管径d、管长l、流速u、密度ρ、粘度μ、管壁粗糙度ε
   - 用因次分析法（π定理）得到无量纲数群：
     - l/d：几何特性
     - Re = ρud/μ：惯性力/黏性力
     - Eu = ΔP/(ρu²)：压力/惯性力
   - 经验公式：
     - **光滑管**：
       - Blasius：λ = 0.3164/Re^0.25（Re=3000~10⁵）
       - 顾毓珍公式（★我国科学家贡献！）：Re=3000~3×10⁶
     - **粗糙管**：
       - Nikurades-Karman公式
       - 顾毓珍公式

7. **Moody摩擦因数图（图1-19）**
   - 横轴：Re（对数坐标）
   - 纵轴：λ（对数坐标）
   - 参数：ε/d（相对粗糙度）
   - **分区**：
     - 层流区：Re≤2000，λ=64/Re，lgλ与lgRe成直线
     - 过渡区：2000<Re<4000，波动
     - 湍流区：Re≥4000，虚线以下，λ随Re减小、随ε/d增大
     - 完全湍流区（阻力平方区）：虚线以上，λ≈常数，h_f∝u²
   - **Colebrook公式**：工程计算用，1/√λ = -2lg[ε/(3.7d) + 2.51/(Re√λ)]

8. **管壁粗糙度**
   - 绝对粗糙度ε：凸出部分的平均高度
   - 相对粗糙度：ε/d
   - 光滑管：玻璃、铝、铜、塑料
   - 粗糙管：钢管、铸铁管

9. **非圆形管**
   - 当量直径：de = 4A/Π = 4×流通截面积/润湿周边长
   - 矩形管（a×b）：de = 2ab/(a+b)
   - 套管环隙：de = d₂ - d₁
   - 注意：Re和h_f中的d用de；但流速用实际流通面积计算
   - 层流时：λ = C/Re，C=57（正方形），C=96（套管环隙）

10. **局部阻力**
    - **阻力系数法**：h_f = ζ·u²/(2g)
    - **当量长度法**：h_f = λ(l_e/d)(u²/2g)
    - **常见ζ**：
      - 突然扩大：ζ=(1-A₁/A₂)²
      - 突然缩小：ζ=0.5（A₂/A₁<0.2时）
      - 管进口：ζ=0.5
      - 管出口：ζ=1
      - 45°弯头：0.35；90°弯头：0.75
      - 截止阀全开：6.0；半开：9.5
    - **管路总阻力**：Σh_f = Σh_f,直管 + Σh_f,局部

### 重点例题

**例1-14：甘油层流（层流计算经典）**
- 20°C甘油：μ=1499mPa·s=1.499Pa·s，ρ=1261
- Q=5×10⁻⁴m³/s，d=27mm=0.027m，求每米管长压降
- **步骤**：
  1. u = Q/A = 5×10⁻⁴/(π×0.027²/4) = 0.873 m/s
  2. Re = ρud/μ = 1261×0.873×0.027/1.499 = 19.8 < 2000 → 层流
  3. λ = 64/Re = 64/19.8 = 3.23
  4. ΔP_f = λ(l/d)(ρu²/2) = 3.23×(1/0.027)×(1261×0.873²/2) = 57.3 kPa/m
- **关键**：层流直接λ=64/Re，不用查图；注意μ单位换算（mPa·s→Pa·s）

**例1-15：湍流阻力（查Moody图经典）**
- u=4.5m/s，d=0.05m，μ=4.46×10⁻⁴Pa·s，ρ=800
- 钢管ε=4.46×10⁻⁵m，l=40m，求阻力损失
- **步骤**：
  1. Re = ρud/μ = 800×4.5×0.05/4.46×10⁻⁴ = 4.04×10⁵ > 4000 → 湍流
  2. ε/d = 4.46×10⁻⁵/0.05 = 8.92×10⁻⁴
  3. 查Moody图：Re=4×10⁵，ε/d≈0.0009 → λ≈0.024
  4. ΔP_f = λ(l/d)(ρu²/2) = 0.024×(40/0.05)×(800×4.5²/2) = 311 kPa
- **关键**：Moody图查λ是考试重点；注意ε/d的计算

**例1-16：管径选择（工程实用）**
- 精馏塔进料：10000kg/h，ρ=960，性质与水接近
- **步骤**：
  1. 选流速u=1.8m/s（查表1-1经验数据）
  2. V_s = m_s/ρ = 10000/(960×3600) = 2.89×10⁻³ m³/s
  3. d = √(4V_s/πu) = √(4×2.89×10⁻³/(π×1.8)) = 0.0452m = 45.2mm
  4. 查附录22水管规格，圆整为φ57×3mm（内径51mm）
  5. 重新核算流速：u = V_s/A = 2.89×10⁻³/(π×0.051²/4) = 1.41 m/s
- **关键**：先选流速→算直径→查规格圆整→核算流速；圆整后流速通常降低

**例1-17：试差法计算管径（★迭代经典）**
- 10°C水：Q=10m³/h=10/3600 m³/s，l=25m，水平管，压差=5mH₂O
- 求适宜管径
- **步骤**：
  1. 查10°C水物性：ρ=999.7，μ=1.307×10⁻³Pa·s
  2. 假设u=2.0m/s（在适宜流速范围内）
  3. d = √(4V_s/πu) = √(4×10/3600/(π×2.0)) = 0.042m = 42mm
  4. 查附录22选φ45×2mm钢管（内径41mm）
  5. 校核：
     - 实际u = 10/3600/(π×0.041²/4) = 2.11 m/s
     - Re = 999.7×2.11×0.041/1.307×10⁻³ = 6.62×10⁴
     - 查新钢管粗糙度ε=0.1mm → ε/d=0.1/41=0.00244
     - 查Moody图：Re=6.6×10⁴，ε/d=0.0024 → λ≈0.03
     - 所需压头h_f = λ(l/d)(u²/2g) = 0.03×(25/0.041)×(2.11²/2×9.81) = 3.18m
     - 给定H_f0=5m > 3.18m → 符合要求
  6. 若不符合，重选管径再校核
- **关键**：试差法先假设u或λ→计算→校核→迭代；若已知在阻力平方区或层流，可直接求解

### 常见陷阱
- d用内径：φ57×3.5mm → 内径50mm，不是57mm
- Moody图读错：Re和ε/d两个参数都要对准
- 局部阻力用错流速：突然扩大用进口（小管）流速
- 非圆形管流速：用实际流通面积计算，不是πde²/4
- 试差法不知道何时停止：通常压头误差<5%即可

## 第一章 1.6 管路计算

### 核心知识点

1. **简单管路**
   - 特点：质量流量不变，不可压缩时体积流量不变
   - 总能量损失 = 各段能量损失之和

2. **复杂管路**
   - 并联管路：各支管压降相等，总流量=各支管之和
   - 分支管路：分支点总压头恒定

3. **管路计算基本方程**
   - 连续性方程：u₁A₁ = u₂A₂
   - 伯努利方程：z₁ + P₁/ρg + u₁²/2g + H_e = z₂ + P₂/ρg + u₂²/2g + H_f
   - 阻力计算：h_f = λ(l/d)(u²/2g)
   - 物性一定时，需给定9个独立参数，可求解3个未知量

4. **常用三种计算**
   - 已知d、l、管件、Q，求能量损失
   - 给定Q、l、管件、允许ΔP，求d（试差法）
   - 已知d、l、允许ΔP，求u或Q（试差法）

## 第一章 1.7 流体输送机械（★★★重点）

### 核心知识点

1. **离心泵工作原理**
   - 灌泵：启动前泵壳内灌满液体
   - 排液：叶轮旋转→离心力→液体从中心→外围→动能→静压能
   - 吸液：中心形成真空→液面压强与泵内压差→液体吸入
   - **核心**：依靠高速旋转叶轮产生的离心力

2. **气缚现象**
   - 泵内充满气体→无法形成足够真空→不能吸液
   - 防止：启动前灌泵；吸入管入口装止逆阀（底阀）
   - 若泵位置低于槽内液面，启动时无需灌泵

3. **主要构造**
   - 叶轮：闭式（干净流体，效率高）、开式（含固体）、半闭式（浆料）
     - 闭式效率最高，开式最低
     - 单吸式 vs 双吸式（双吸可消除轴向推力）
   - 泵壳（蜗壳）：汇集液体，动能→静压能
   - 导轮：减少碰撞，引导液体平缓改变方向
   - 轴封装置：填料密封（普通）或机械密封（端面密封）

4. **主要性能参数**
   - 流量Q（m³/s或m³/h）
   - 扬程H（m液柱）：泵对单位质量流体提供的有效压头
   - 效率η：50%~70%（小型泵），可达90%以上（大型泵）
     - 容积损失：泄漏
     - 水力损失：涡流、冲击
     - 机械损失：轴承摩擦
   - 轴功率N = ρgQH/(1000η)（kW，Q用m³/s）
   - 电机功率按(1.1~1.2)N选取，防止超负荷

5. **特性曲线**
   - H-Q：流量增大，扬程下降
   - N-Q：流量增大，轴功率上升；流量为零时N最小
     - **启动时关闭出口阀**：保护电机
   - η-Q：有一最高效率点
     - 铭牌参数就是最高效率点的Q、H、N
     - 操作效率应不低于最高效率的92%

6. **液体性质影响**
   - 密度：H与ρ无关，N与ρ成正比
   - 粘度：μ>20cSt时，H↓、Q↓、η↓、N↑；需修正特性曲线

7. **比例定律**
   - Q₁/Q₂ = n₁/n₂
   - H₁/H₂ = (n₁/n₂)²
   - N₁/N₂ = (n₁/n₂)³
   - 适用：粘度不大，转速变化<20%

8. **汽蚀现象（★★★必考）**
   - 产生条件：叶片入口附近p_k ≤ p_v（饱和蒸气压）
   - 过程：液体汽化→气泡进入高压区→迅速凝结→局部真空→周围液体高速冲击→震动、噪音、腐蚀
   - 后果：
     - 泵体震动、噪音
     - 叶片蜂窝状腐蚀
     - 流量、压头、效率明显下降
     - 严重时吸不上液体
   - 避免条件：p_k > p_v

9. **安装高度（★★★必考）**
   - 定义：泵入口中心线与吸入槽液面之间的垂直距离
   - 在液面0与泵入口1间列伯努利方程推导：
     ```
     H_g = P_0/(ρg) - P_1/(ρg) - u_1²/(2g) - H_f,吸入
     ```
   - 当p_k = p_v时，刚好发生汽蚀，对应H_g,max
   - **允许汽蚀余量NPSH**：
     ```
     NPSH_r = (P_1/(ρg) + u_1²/(2g)) - P_v/(ρg)
     ```
     - 泵入口处液体的静压头+动压头之和超出饱和蒸气压头的最小值
     - 由厂家实验测定，在大气压10mH₂O、20°C清水下测得
     - 流量越大，NPSH_r越大
   - **安装高度计算公式**：
     ```
     H_g = P_0/(ρg) - P_v/(ρg) - NPSH_r - H_f,吸入
     ```
     - 敞开贮槽：P_0 = P_a（大气压）
     - 安全裕量：0.5m（极限值→允许值）
   - **提高安装高度的途径**：
     - 减小吸入管路阻力
     - 较大的吸入管径
     - 管路尽可能短
     - 减少弯头、阀门，调节阀装在排出管线上

10. **工作点与流量调节**
    - 泵特性曲线：H = f(Q)，由泵本身决定
    - 管路特性曲线：H_e = Δz + ΔP/(ρg) + BQ²
    - 工作点：两曲线交点
    - **调节出口阀**：
      - 关小→管路阻力增大→管路曲线变陡→工作点左移→Q↓、H↑
      - 优点：简捷方便，流量连续变化，适用于经常性调节
      - 缺点：不经济，阀门关小压头增高，能量损失增大
    - **改变转速**：
      - 提高转速→泵特性曲线上移→工作点右上移→Q↑、H↑
      - 优点：动力消耗降低，不额外增加管路阻力，保持高效率
      - 缺点：需要变速装置，难以连续调节，化工中很少采用

11. **泵的组合**
    - 串联：在同样流量下，压头为单泵的两倍
      - 单泵流量<串联流量<两泵独立流量之和
    - 并联：在同样压头下，流量为单泵的两倍
      - 单泵流量<并联流量<两泵独立流量之和
    - **选择**：
      - 低阻输送管路：并联优于串联
      - 高阻输送管路：串联优于并联
      - 若管路所需压头大于单泵最大压头，必须串联

12. **泵的类型**
    - 清水泵：IS型（单级单吸）、D型（多级）、Sh型（双吸）
    - 耐腐蚀泵：F型
    - 油泵：Y型
    - 杂质泵：P型（污水、砂、泥浆）
    - 磁力泵：无轴封，无渗漏，安全
    - 屏蔽泵：无密封泵，输送易燃、易爆、剧毒液体

13. **泵的选型**
    - 步骤：
      1. 选择类型（根据液体性质）
      2. 确定Q和H（按最大流量考虑，用伯努利方程计算H_e）
      3. 查样本选型号（选H和Q都稍大的，考虑效率）
      4. 核算轴功率（若密度>水，N需重新计算）

14. **启动与关闭步骤**
    - 启动：灌泵→关出口阀→启动电机→逐渐开出口阀
    - 关闭：关出口阀→停电机→关入口阀
    - 注意：先关出口阀再停电机，防止高压液体反冲入泵

### 重点例题

**例1-18：泵安装高度计算（★★必考大题）**
- IS65-40-200泵，n=2900r/min，Q=25m³/h，H=50m，NPSH_r=2.0m
- 输送50°C水，敞口水池，吸入管路总阻力H_f=2m
- 当地大气压P_a=100kPa
- **步骤**：
  1. 查50°C水物性：P_v=12.31kPa，ρ=998.1kg/m³
  2. H_g = P_a/(ρg) - P_v/(ρg) - NPSH_r - H_f
     = 100000/(998.1×9.81) - 12310/(998.1×9.81) - 2.0 - 2.0
     = 10.21 - 1.26 - 2.0 - 2.0 = 4.95 m
  3. 考虑安全裕量0.5m：H_g,允许 = 4.95 - 0.5 = 4.45 m
- **关键**：
  - 必须查50°C水的饱和蒸气压（P_v=12.31kPa），不能忽略！
  - NPSH_r用最高流量下的值
  - 安全裕量0.5m必须减
  - 若海拔高（P_a低）或温度高（P_v高），安装高度会大幅降低

### 常见陷阱
- 功率公式中Q单位：Q用m³/s（25m³/h = 25/3600 m³/s），不是m³/h
- 比例定律适用范围：同一泵、同一液体、转速变化<20%
- 安装高度与海拔/温度：标准安装高度是海平面、20°C；实际必须修正
- NPSH_a vs NPSH_r：a=available（实际），r=required（泵要求）
- 启动前灌泵：离心泵必须关出口阀启动；往复泵应开出口阀启动
- 泵的选型时，若密度大于水，轴功率需重新计算

## 2.1 传热基本概念

### 核心知识点

1. **化工生产中的传热要求**
   - **强化传热**：各种换热设备中的传热（需要增大K、S或Δtm）
   - **削弱传热**：设备、管道的保温，尽量减少热损失（加保温层）

2. **三种传热方式**
   - **热传导**：直接接触的系统之间或系统内各部分之间，无宏观相对运动，仅借分子、原子及自由电子等微观粒子热运动传递热量
   - **对流传热**：流体各部分之间发生相对位移所引起的热量传递，仅发生在流体中，伴随流体流动
   - **热辐射**：物体由于热的原因而发出辐射能的过程，不需要介质，可在真空中传播

## 2.2 热传导（★★★重点）

### 核心知识点

1. **温度场与等温面**
   - 温度场：某时刻物体内各点温度分布，t = f(x,y,z,θ)
   - 等温面：同一时刻温度相同的各点组成的面，不会相交

2. **傅里叶定律（热传导基本定律）**
   ```
   dQ = -λ·S·(dt/dx)
   ```
   - Q：热传导速率（W），方向与温度梯度相反
   - λ：热导率（导热系数），W/(m·℃)，表征物质导热能力
   - S：与热传导方向垂直的导热面面积（m²）
   - 一般规律：金属 > 非金属固体 > 液体 > 气体

3. **单层平壁稳态热传导**
   ```
   Q = λ·S·(t₁-t₂)/b = Δt/R = 传热推动力/传热阻力
   ```
   - 热阻 R = b/(λS) （℃/W）
   - 导热通量（热流密度）q = Q/S = λ·(t₁-t₂)/b
   - **温度分布是直线**：t = t₁ - (Q·x)/(λS)

4. **多层平壁稳态热传导（★★★必考）**
   ```
   Q = (t₁-tₙ₊₁) / Σ(bᵢ/(λᵢS)) = 总推动力/总热阻
   ```
   - **串联热阻**：总热阻 = 各层热阻之和
   - **温度差分配**：Δt₁ : Δt₂ : Δt₃ = R₁ : R₂ : R₃ = b₁/λ₁ : b₂/λ₂ : b₃/λ₃
   - **关键结论**：哪层热阻大，哪层温度差就大；反之亦然
   - 接触热阻：实际两层接触处因粗糙不平留有空穴，热阻变大

5. **单层圆筒壁稳态热传导**
   ```
   Q = 2πλL(t₁-t₂) / ln(r₂/r₁) = (t₁-t₂) / (b/(λ·Sₘ))
   ```
   - Sₘ = 2πL·rₘ，rₘ = (r₂-r₁)/ln(r₂/r₁) = 对数平均半径
   - Sₘ = 2πL·rₘ = 对数平均面积
   - **温度分布是对数曲线**：t ~ ln(r) 成线性关系
   - 不同半径处Q相等，但q不等（q = Q/(2πrL)，r越大q越小）

6. **多层圆筒壁稳态热传导**
   ```
   Q = (t₁-tₙ₊₁) / Σ[ln(rᵢ₊₁/rᵢ)/(2πλᵢL)]
   ```
   - 各层用各自的对数平均面积
   - 推广至n层：总推动力/总热阻

### 重点例题

**例2-1：单层平壁热传导（求q）**
- 直接套用 q = λ·Δt/b
- 注意单位统一：λ用W/(m·K)，b用m，Δt用K或℃

**例2-2：多层平壁（炉墙散热，★经典）**
- 耐火砖λ₁=1.09，硅藻土λ₂=0.116，金属板λ₃=45
- b₁=115mm, b₂=185mm, b₃=3mm；t₁=642℃, t₄=54℃
- **求（1）单位面积散热速率q**：
  ```
  q = (t₁-t₄) / (b₁/λ₁ + b₂/λ₂ + b₃/λ₃)
    = (642-54) / (0.115/1.09 + 0.185/0.116 + 0.003/45)
    = 588 / (0.1055 + 1.5948 + 0.00007) = 346 W/m²
  ```
- **求（2）耐火砖与硅藻土交界处温度t₂**：
  ```
  q = (t₁-t₂)/(b₁/λ₁) → t₂ = t₁ - q·b₁/λ₁ = 642 - 346×0.115/1.09 = 606℃
  ```
- **关键**：金属板热阻极小，温差几乎为零；硅藻土热阻最大，温差最大

**例2-3：多层圆筒壁（管路保温，★经典）**
- 管内径160mm，外径170mm；内层保温20mm，外层40mm
- λ₁=58.2（钢管），λ₂=0.174，λ₃=0.093；t₁=300℃，t₄=50℃
- **求每米管长热损失Q/L**：
  ```
  r₁=80mm, r₂=85mm, r₃=105mm, r₄=145mm
  Q/L = 2π(t₁-t₄) / [ln(r₂/r₁)/λ₁ + ln(r₃/r₂)/λ₂ + ln(r₄/r₃)/λ₃]
  ```
- **关键**：圆筒壁用对数平均，各层半径要算对

### 常见陷阱
- 温度分布：平壁是直线，圆筒壁是对数曲线
- 多层壁温度差分配：热阻大的层温差大，不要想当然平均分配
- 圆筒壁的q：不同半径处q不同，但Q相同
- 接触热阻：实际工程中不可忽视，但考试通常忽略
- 单位：b必须用m，λ用W/(m·K)，结果才是W

## 2.3 对流传热（★★★重点）

### 核心知识点

1. **对流传热机理**
   - 湍流流体流经固体壁面时，壁面处热量以热传导方式进入层流内层，再进入湍流核心
   - **层流内层是主要热阻**：虽薄但λ小，温度降主要集中在这里
   - 强化对流传热 → 减薄层流内层 → 增大湍动程度

2. **牛顿冷却定律（★核心公式）**
   ```
   Q = α·S·Δt
   ```
   - α：对流传热系数（W/(m²·℃)），反映对流传热强度
   - α数值上等于单位温度差、单位时间、单位面积以对流方式传递的热量
   - **牛顿冷却定律实质**：将矛盾集中到α，研究α成为核心

3. **影响α的因素**（量纲分析得到特征数）
   - 流体种类、相变化、物理性质（ρ,μ,λ,Cp）
   - 流动状况（层流/湍流，湍动程度）
   - 流动原因（自然对流 vs 强制对流）
   - 传热面形状、位置、大小

4. **特征数关联式**
   ```
   Nu = f(Re, Pr, Gr)
   ```
   - **Nu = α·l/λ**（努塞尔数，反映对流与纯导热的比值）
   - **Re = ρul/μ**（雷诺数，反映流动状态）
   - **Pr = Cp·μ/λ**（普朗特数，反映流体物性）
   - **Gr = βgΔtl³ρ²/μ²**（格拉晓夫数，反映自然对流强度）

5. **无相变时管内强制对流（★考试重点）**
   - **低黏度流体湍流（Dittus-Boelter关联式）**：
     ```
     Nu = 0.023·Re^0.8·Pr^n
     ```
     - 液体被加热：n=0.4；液体被冷却：n=0.3
     - 气体：n=0.4（加热），n=0.3（冷却）——虽然气体黏度随温度变化趋势与液体相反，但Pr<1使Pr^0.4>Pr^0.3
     - 适用范围：Re>10⁴，0.7<Pr<120，L/di>60
     - 定性温度：流体进出口平均温度；特性尺寸：管内径di
   - **高黏度流体（Sieder-Tate关联式）**：
     ```
     Nu = 0.027·Re^0.8·Pr^(1/3)·(μ/μw)^0.14
     ```
     - 校正项：液体被加热取1.05，冷却取0.95，气体取1.0
   - **层流（Sieder-Tate）**：
     ```
     Nu = 1.86·(Re·Pr·di/L)^(1/3)·(μ/μw)^0.14
     ```
     - 适用范围：Re<2300，(Re·Pr·di/L)>100

6. **无相变时管外强制对流**
   - 管束外垂直流动：
     - 直列：Nu = 0.26·Re^0.6·Pr^0.3
     - 错列：Nu = 0.33·Re^0.6·Pr^0.3
     - 特性尺寸：管外径d₀；流速：最狭窄通道处速度
   - 管壳式换热器壳程（Donohue法或Kern法）
     - 当量直径de（正方形/正三角形排列公式不同）

7. **有相变时的对流传热（α大得多）**
   - **蒸汽冷凝**：
     - 膜状冷凝：冷凝液润湿壁面形成液膜，热阻主要在液膜；α_膜 < α_滴
     - 滴状冷凝：冷凝液不润湿壁面成滴落下；α_滴 > α_膜（可达5~10倍）
     - 竖壁/竖管层流：α = 1.13·(ρ²gλ³r/(μLΔt))^(1/4)
     - 竖壁/竖管湍流：α = 0.0077·(ρ²gλ³/(μ²))^(1/3)·Re^0.4
     - 水平管束外：α = 0.725·(ρ²gλ³r/(μd₀Δt))^(1/4)
     - **影响因素**：不凝气体（α↓↓，1%空气降60%）、蒸汽流速、蒸汽过热、液膜厚度
   - **液体沸腾**：
     - 大容积沸腾 vs 管内沸腾（强制对流沸腾）
     - 沸腾曲线：自然对流→核状沸腾（泡状沸腾）→过渡区→膜状沸腾
     - **临界点**：核状沸腾→膜状沸腾转折点，工程上控制在核状沸腾区操作
     - 强化：加表面活性剂、提高温差（核状区）、提高压强、新粗糙加热面

8. **使用α关联式注意事项**
   - 先判断问题类型：有无相变、强制/自然对流、管内/管外
   - 注意应用范围：Re、Pr范围
   - 注意特性尺寸：Nu、Re中的l如何确定
   - 注意定性温度：取流体平均温度、壁温或膜温
   - **数量级**：有相变 > 无相变；强制 > 自然；水 > 空气

### 重点例题

**例2-4/例2-5：对流传热系数α计算（P85）**
- 直接通过实验求得计算α的关联式
- 注意：非圆形管de = 4×流通截面积/润湿周边

### 常见陷阱
- 定性温度取错：多数取进出口平均温度，壁温取μw
- 特性尺寸用错：管内用di，管外用d₀，管束用de
- 热流方向（加热/冷却）的n值取反：液体加热n=0.4，冷却n=0.3
- 有相变时α大得多，不要与无相变混淆
- 层流底层热阻控制：α大≠K大，还要看另一侧

## 2.4 热辐射

### 核心知识点

1. **热辐射特点**
   - 热能→辐射能转化，温度不变则辐射能不变
   - 高温→低温传递能量，物体同时发射和吸收辐射能
   - 波长在0.4~40μm（可见光+红外线）
   - 不需要介质，可在真空中传播

2. **吸收、反射、透过**
   - A + R + D = 1（吸收率+反射率+透过率）
   - 固体/液体：D≈0，A+R=1
   - 气体：R≈0，A+D=1

3. **黑体、白体、灰体**
   - **黑体**：A=1，理想物体（无光泽黑漆表面A≈0.97）
   - **白体（镜体）**：R=1，磨光金属表面R≈0.97
   - **灰体**：吸收率A与波长无关，大多数工程材料可视为灰体

4. **斯蒂芬-玻尔兹曼定律（★核心公式）**
   ```
   E₀ = C₀·(T/100)⁴
   ```
   - E₀：黑体辐射能力（W/m²）
   - C₀ = 5.67 W/(m²·K⁴)：黑体辐射系数
   - **四次方定律**：热辐射对温度特别敏感！

5. **灰体辐射能力**
   ```
   E = ε·E₀ = ε·C₀·(T/100)⁴ = C·(T/100)⁴
   ```
   - ε：黑度（发射率），0<ε<1，C = ε·C₀
   - **克希霍夫定律**：A = ε（同一温度下，吸收率=黑度）
   - 黑度可实验测定，吸收率不易测，工程中用黑度代替吸收率

6. **两灰体间辐射传热**
   ```
   Q = C₁₋₂·φ·S·[(T₁/100)⁴ - (T₂/100)⁴]
   ```
   - C₁₋₂：总辐射系数，与两物体黑度有关
   - φ：角系数（几何因数），从辐射面积S发射的能量被另一表面截获的分数
   - 必须和选定的辐射面积S相对应

7. **对流与辐射联合传热**
   ```
   Q = αT·S·(tw - t)
   ```
   - αT：对流辐射联合传热系数
   - 空气自然对流（tW<150℃）：平壁αT = 9.8 + 0.07(tw-t)；管道αT = 9.4 + 0.052(tw-t)
   - 空气强制对流：u≤5m/s时αT = 5.3 + 3.6u；u>5m/s时αT = 6.7u^0.78

### 常见陷阱
- 温度必须用绝对温度（K）！不能用℃
- 四次方关系：温度翻倍，辐射能力增16倍
- 角系数φ和面积S必须对应
- 灰体A=ε仅在同一温度下成立

## 2.5 间壁式换热器传热计算（★★★★★核心）

### 核心知识点

1. **间壁式换热过程**
   - 热流体主体→对流→间壁内侧→导热→间壁外侧→对流→冷流体主体
   - 三个串联步骤：对流+导热+对流

2. **热量衡算**
   - **无相变**：Q = Wh·Cph·(T₁-T₂) = Wc·Cpc·(t₂-t₁)
   - **一侧有相变（饱和蒸汽冷凝）**：Q = Wh·r = Wc·Cpc·(t₂-t₁)
   - **有相变+显热变化**：Q = Wh·r + Wh·Cph·(Td-T₂) = Wc·Cpc·(t₂-t₁)
   - 考虑热损失：Q_放 = Q_吸 + Q_损

3. **总传热速率方程（★★★核心）**
   ```
   Q = K·S·Δtm
   ```
   - K：总传热系数（W/(m²·℃)），反映传热过程强度
   - S：传热面积（m²），工程上习惯以管外面积为基准
   - Δtm：平均温度差（℃）

4. **总传热系数K的确定**
   - **理论计算法**：
     ```
     1/K₀ = 1/α₀ + b·d₀/(λ·dm) + d₀/(αᵢ·dᵢ) + Rso + Rsi·d₀/dᵢ
     ```
     - 以管外面积为基准：1/K₀ = 1/α₀ + (b/λ)·(d₀/dm) + (1/αᵢ)·(d₀/dᵢ) + Rso + Rsi·(d₀/dᵢ)
     - 以内表面积为基准：1/Kᵢ = dᵢ/(α₀·d₀) + (b/λ)·(dᵢ/dm) + 1/αᵢ + Rso·(dᵢ/d₀) + Rsi
     - dm = (d₀-dᵢ)/ln(d₀/dᵢ) = 对数平均直径
   - **污垢热阻（★考试重点）**：
     - Rso、Rsi：管外侧和内侧污垢热阻
     - 通常污垢热阻比传热壁热阻大得多，设计中不可忽视
     - P98表2-7给出常用污垢热阻经验值
   - **K值讨论（★重要结论）**：
     - 当α₀<<αᵢ时，K≈α₀；当αᵢ<<α₀时，K≈αᵢ
     - **K总是接近α较小的一侧**（热阻大的一侧控制）
     - 提高K值：关键在于提高α较小一侧的α值
     - 若两侧α都很大，污垢热阻控制→减慢污垢形成或及时清除
     - K值比两侧α中较小者还小

5. **平均温度差Δtm（★★★必考）**
   - **恒温传热**：Δtm = T - t（常数）
   - **变温传热**：
     - **逆流/并流**：
       ```
       Δtm = (Δt₁ - Δt₂) / ln(Δt₁/Δt₂)
       ```
       - Δt₁、Δt₂：换热器两端两流体的温度差（较大者记为Δt₁）
       - 当Δt₁/Δt₂ < 2时，可用算术平均值代替（误差<4%）
       - 当Δt₁ = Δt₂时，Δtm = Δt₁ = Δt₂
     - **逆流 vs 并流比较**：
       - 进出口温度相同：Δtm,逆流 > Δtm,并流
       - 工业上多采用逆流（节省面积或降低介质流量）
       - 只有冷流体被加热/热流体被冷却不允许超温时，用并流更可靠
     - **错流/折流**：
       ```
       Δtm = Δtm,逆流 × φΔt
       ```
       - φΔt：温差校正系数，查P118图2-39
       - 一般设计时φΔt > 0.8

6. **换热器传热计算类型**
   - **设计型计算**：根据热负荷求传热面积S
   - **操作型计算**：对现有换热器判断适用性或预测参数变化影响
   - **计算复杂，常需试差或迭代**

7. **强化传热途径**
   - 增大推动力：Δtm↑（逆流、降低冷却剂温度/提高加热剂温度）
   - 增大传热面积：S↑（直接接触、加翅片）
   - 减小热阻：K↑（提高α较小一侧、减薄壁厚、及时除垢）

8. **传热效率-传热单元数法（ε-NTU法，★难点）**
   - **传热效率**：ε = Q/Qmax = 实际传热量/最大可能传热量
   - Qmax = (WCp)min·(T₁-t₁)（最小热容流率流体×最大温差）
   - **传热单元数**：NTU = K·S/(WCp)min
   - **优点**：
     - 计算简单，查图方便
     - 两流体出口温度均未知时，无须试差
     - 改变进口温度可直接求出口温度
     - 串联换热器可将NTU累计
   - 关系：ε = f(NTU, Cmin/Cmax, 流动方式)，查P106图2-19

### 重点例题

**例2-7：总传热系数K计算（★经典）**
- φ25×2.5mm钢管，αᵢ=50（CO₂侧），α₀=2500（水侧）
- λ=45，Rsi=0.0005，Rso=0.00058
- **求（1）K₀**：
  ```
  1/K₀ = 1/α₀ + b·d₀/(λ·dm) + d₀/(αᵢ·dᵢ) + Rso + Rsi·d₀/dᵢ
       = 1/2500 + 0.0025×25/(45×22.5) + 25/(50×20) + 0.00058 + 0.0005×25/20
       = 0.0004 + 0.0000617 + 0.025 + 0.00058 + 0.000625 = 0.02667
  K₀ = 37.5 W/(m²·K)
  ```
- **求（2）αᵢ提高一倍**：K₀' = 70.6，增大88.2%
- **α₀提高一倍**：K₀'' = 37.8，仅增大0.8%
- **关键结论**：K由热阻大的一侧（αᵢ=50）控制，提高α小侧效果显著

**例2-8：并流/逆流平均温度差计算（★基础）**
- 热水：360K→340K；冷料液：290K→330K
- **并流**：Δt₁=70K，Δt₂=10K，Δtm = (70-10)/ln(70/10) = 30.8K
- **逆流**：Δt₁=50K，Δt₂=30K，Δtm = (50-30)/ln(50/30) = 39.2K
- **逆流Δtm > 并流Δtm**

**例2-9：换热器设计型计算（★★大题）**
- 套管换热器：苯65℃→15℃（1kg/s），冷却水10℃→45℃（逆流）
- φ25×2.5mm内管，α苯=0.82×10³，α水=1.7×10³
- Cp苯=1.88×10³，Cp水=4.18×10³，λ钢=45
- **求（1）冷却水消耗量**：
  ```
  Q = Wh·Cp苯·(T₁-T₂) = 1×1.88×10³×(65-15) = 94×10³ W
  Wc = Q/[Cp水·(t₂-t₁)] = 94×10³/[4.18×10³×(45-10)] = 0.643 kg/s
  ```
- **求（2）总管长**：
  ```
  Δtm = [(65-45)-(15-10)]/ln[(65-45)/(15-10)] = (20-5)/ln(20/5) = 10.8℃
  1/K₀ = 1/1700 + 0.0025×25/(45×22.5) + 25/(820×20) = 0.00202
  K₀ = 496 W/(m²·K)
  S = Q/(K₀·Δtm) = 94×10³/(496×10.8) = 17.5 m²
  L = S/(π·d₀) = 17.5/(π×0.025) = 223 m
  ```
- **关键**：热量衡算→求Q→求Δtm→求K→求S→求L；注意K以管外为基准

**例2-10：操作型计算（水量变化，★★试差）**
- 原工况：冷水0.6kg/s（30℃），空气2.52kg/s（130℃→70℃）
- α水=2000，α空气=50，Cp水=4200，Cp空气=1000
- **求水量增加1倍后Q'/Q**：
  - 原K₀：1/K₀ = 1/2000 + 1/50 → K₀≈48.8（空气侧控制）
  - 水量加倍→α水变为2^0.8×2000≈3480，但K₀仍≈49.3（空气侧控制，几乎不变）
  - 热量衡算+传热速率方程联立→试差求解新出口温度→Δtm'→Q'/Q≈1.15
  - **结论**：增加热阻较小侧流量，K变化很小，Q增加主要靠Δtm增大

**例2-11：ε-NTU法（★难点）**
- 逆流：热流体2.85kg/s，110℃，Cp=1.9；冷水0.667kg/s，35℃，Cp=4.18
- S=15.8m²，K=320
- **求出口温度和传热量**：
  - (WCp)h = 2.85×1.9×10³ = 5415 W/K
  - (WCp)c = 0.667×4.18×10³ = 2788 W/K（最小热容流率）
  - Cmin/Cmax = 2788/5415 = 0.515
  - NTU = K·S/(WCp)min = 320×15.8/2788 = 1.81
  - 查图：ε = 0.73
  - Q = ε·(WCp)min·(T₁-t₁) = 0.73×2788×(110-35) = 153 kW
  - t₂ = t₁ + Q/(WCp)c = 35 + 153000/2788 = 89.8℃
  - T₂ = T₁ - Q/(WCp)h = 110 - 153000/5415 = 81.8℃

### 常见陷阱
- K的基准面积：工程上以管外为基准，注意公式中d₀/dᵢ的换算
- 污垢热阻不可忽略：实际K比理论值小很多
- 对数平均温差：Δt₁和Δt₂必须是换热器两端的温差，不是进出口温度本身
- Δt₁/Δt₂<2时可用算术平均，但考试通常要求精确计算
- 操作型计算需联立热量衡算和传热速率方程，常需试差
- ε-NTU法中(WCp)min是热容流率，不是流量

## 2.6 换热设备

### 核心知识点

1. **换热器类型**
   - **直接接触式（混合式）**：凉水塔、洗涤塔，冷热能直接混合
   - **蓄热式**：填充物交替吸热放热，高温气体余热利用
   - **间壁式**：
     - 管式：套管式、蛇管式、管壳式（列管式）
     - 板式：螺旋板式、平板式、板翅式
     - 翅片式：板翅式、翅片管式（用于气体换热）
     - 热管换热器

2. **管壳式换热器（最常用）**
   - **固定管板式**：结构简单，但壳程不能清洗，温差大时产生热应力
   - **浮头式**：管束可自由伸缩，壳程可清洗，结构复杂造价高
   - **U形管式**：结构简单，耐高温高压，但管内清洗困难
   - **折流挡板**：引导流体横向流过管束，增强传热，支撑管束

3. **强化传热途径**
   - 增大Δtm：逆流、提高加热剂温度/降低冷却剂温度
   - 增大S：加翅片、直接接触
   - 增大K：提高α较小一侧、减薄壁厚、及时除垢
   - 提高α方法：增大流速、加扰流元件、改变传热面形状

4. **管壳式换热器设计/选用步骤**
   - 估算传热面积，初选换热器类型
   - 计算传热量、确定流体流动途径和进出口温度
   - 计算定性温度、物性参数、平均温度差
   - 确定壳程数（φΔt>0.8）和换热器型式
   - 选取总传热系数K（经验值），初算传热面积
   - 确定基本尺寸或选标准规格
   - 计算管壳程压降
   - 校核K计/K选=1.15~1.25

5. **流体流动通道选择原则**
   - 不清洁、易结垢、有腐蚀性、高压流体→走管程
   - 饱和蒸汽、被冷却的、α大、流量小、黏度大的流体→走壳程
   - 饱和蒸汽走壳程原因：干净、冷凝水易排出、α大不需增大流速、接触壳体和管子热应力小

### 常见陷阱
- 管壳式换热器选用时K计/K选应在1.15~1.25之间
- 温差校正系数φΔt一般应>0.8
- 管子规格：我国标准φ19×2mm和φ25×2.5mm两种
- 管长取6m的倍数（1.5、2、3、6m）
- 正三角形排列紧凑但清洗困难，正方形排列松散但清洗容易
