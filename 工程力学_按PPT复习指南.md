# 工程力学按 PPT 复习指南

> 基于你提供的 11 个 PDF/PPT 材料整理。  
> 注意：当前材料是课件、专题讲义和习题，不是历年试卷，所以“可能考”表示基于课件小结、例题、习题出现频率推断出的高优先级复习点。

## 1. 先看整体：这门课的知识架构

这批资料其实分成三层：

```text
第一层：静力学建模
Lllx1 -> Lllx2 -> Lllx3
作用：会画受力图、列平衡方程、求支座反力和杆件受力。

第二层：材料力学基本语言
Cllx0 -> Cllx1 -> Cllx2
作用：知道强度、刚度、稳定是什么；会用应力、应变、内力、截面几何量描述构件。

第三层：考试高频专题
弯曲内力 -> 弯曲正应力 -> 弯曲强度/刚度/变形 -> 压杆稳定
作用：画剪力图弯矩图，做弯曲强度校核，算挠度转角，做压杆稳定校核。
```

更像老师会考的主线是：

```text
受力分析
  -> 求支座反力
  -> 截面法求内力
  -> 画 Fs 图和 M 图
  -> 找危险截面 / 最大弯矩
  -> 用 σ = M/W 做强度校核
  -> 用 EIw'' = M(x) 做变形计算
  -> 对受压杆件用 Fcr 做稳定校核
```

## 3. 按文件逐个看

## Lllx1.pdf：理论力学第一章，静力学基础

### 这个 PPT 大概讲什么

这是整门课的力学语言入门。主要讲：

- 力的概念。
- 静力学基本公理。
- 力矩和力偶的基本思想。
- 常见约束及约束反力。
- 受力分析和受力图。

### 必须掌握的知识点

- 力是矢量，有大小、方向、作用点。
- 二力平衡条件。
- 加减平衡力系原理。
- 力的可传性。
- 力矩：力对点产生转动效应。
- 力偶：大小相等、方向相反、不共线的一对力。
- 常见约束：
  - 光滑接触面：反力沿法线。
  - 柔索：只能拉，沿绳方向。
  - 铰支座：通常有两个反力分量。
  - 滚动支座：通常只有一个垂直于支承面的反力。
  - 固定端：有水平反力、竖向反力和固定端弯矩。

### 可能会怎么考

- 给一个结构图，让你画受力图。
- 判断某个约束反力方向是否正确。
- 给一个杆件或梁，让你先标出所有外力和约束反力。
- 选择题问“这个约束能提供几个未知反力”。

### 易错点

- 漏画固定端弯矩。
- 把滚动支座画成两个反力。
- 柔索方向画反。
- 受力图把内力和外力混在一起。

## Lllx2.pdf：理论力学第二章，平面汇交力系

### 这个 PPT 大概讲什么

它讲多个力作用线交于一点时，如何合成、分解和平衡。它是后面求杆件节点受力、求简单支座反力的基础。

### 必须掌握的知识点

- 力的分解：`Fx = F cos α`，`Fy = F sin α`。
- 合力投影：
  - `Rx = ΣFx`
  - `Ry = ΣFy`
  - `R = sqrt((ΣFx)^2 + (ΣFy)^2)`
- 平面汇交力系平衡条件：
  - `ΣFx = 0`
  - `ΣFy = 0`
- 力对点之矩：`M = Fd`。
- 合力矩定理。
- 力偶矩与力偶等效。

### 可能会怎么考

- 给斜杆或斜向力，让你分解成水平、竖向分量。
- 给汇交力系，求未知力。
- 给一个简单结构，要求某点力矩。
- 判断力偶矩方向或大小。

### 易错点

- `sin` 和 `cos` 对应边搞反。
- 斜杆受力方向不沿杆轴。
- 求矩时力臂取错，不是随便拿一段长度乘力。
- 力偶可以平移，单个力不能随便平移。

## Lllx3.pdf：理论力学第三章，平面一般力系

### 这个 PPT 大概讲什么

这是静力学里最直接服务考试计算题的一章。它讲不共点力系如何简化、如何列平衡方程、如何处理物体系平衡和摩擦。

### 必须掌握的知识点

- 力线平移定理：平移一个力，需要附加一个力偶。
- 平面一般力系向一点简化。
- 主矢和主矩。
- 平面一般力系平衡方程：
  - `ΣFx = 0`
  - `ΣFy = 0`
  - `ΣM = 0`
- 分布载荷可以等效为集中力，作用在载荷图形形心处。
- 物体系平衡：可以整体分析，也可以拆开分析。
- 摩擦力方向总是阻碍相对运动趋势。

### 可能会怎么考

- 给梁，求支座反力。
- 给组合结构，拆开求某个杆件受力。
- 给分布载荷，先等效再列平衡方程。
- 给含摩擦的问题，判断是否滑动或求临界力。

### 易错点

- 分布载荷合力大小算对了，作用点放错。
- 求矩点选得不好，导致方程很复杂。
- 物体系题只看整体，忘了拆开后有内力。
- 摩擦方向画成“运动方向”，而不是“阻碍运动趋势方向”。

## Cllx0.pdf：材料力学绪论

### 这个 PPT 大概讲什么

它告诉你材料力学到底研究什么。关键词是构件在外力作用下的强度、刚度、稳定性。

### 必须掌握的知识点

- 强度：构件抵抗破坏的能力。
- 刚度：构件抵抗变形的能力。
- 稳定性：构件保持原有平衡形式的能力。
- 材料力学的基本假设：
  - 连续性。
  - 均匀性。
  - 各向同性。
  - 小变形。
- 外力、内力。
- 截面法。
- 应力：
  - 正应力 `σ`
  - 切应力 `τ`
- 应变：
  - 线应变
  - 切应变
- 基本变形形式：
  - 轴向拉伸/压缩。
  - 剪切。
  - 扭转。
  - 弯曲。

### 可能会怎么考

- 选择题：强度、刚度、稳定性的区别。
- 判断题：小变形假设的含义。
- 简答题：材料力学四个基本假设。
- 概念题：正应力和切应力的区别。

### 易错点

- 把稳定性理解成“强度够不够”。
- 把刚度和强度混为一谈。
- 不知道材料力学公式通常建立在小变形、线弹性等前提下。

## Cllx1.pdf：轴向拉压与剪切

### 这个 PPT 大概讲什么

它讲杆件受轴向力时的内力、应力、变形和强度条件，也讲剪切、挤压等连接件问题。

### 必须掌握的知识点

- 轴力图。
- 轴向拉压正应力：`σ = FN/A`。
- 线应变：`ε = Δl/l`。
- 胡克定律：`σ = Eε`。
- 拉压变形：`Δl = FN l / EA`。
- 泊松比。
- 材料拉伸时的力学性能：
  - 弹性阶段。
  - 屈服。
  - 强化。
  - 颈缩。
  - 断裂。
- 许用应力和强度条件：
  - `σmax <= [σ]`
- 剪切和挤压强度条件。

### 可能会怎么考

- 给杆件受力，画轴力图。
- 求某段杆的正应力或总伸长量。
- 给许用应力，校核杆件强度。
- 螺栓、铆钉、销钉连接的剪切和挤压校核。
- 判断低碳钢拉伸曲线各阶段。

### 易错点

- 轴力正负号乱。
- 多段杆变形没有分段相加。
- 剪切面积和挤压面积用错。
- 把材料强度极限、屈服极限、许用应力混用。

## Cllx2.pdf：扭转

### 这个 PPT 大概讲什么

它讲圆轴扭转时的内力、切应力、扭转角和强度/刚度校核。

### 必须掌握的知识点

- 外力偶矩和扭矩。
- 扭矩图。
- 圆轴扭转切应力分布：外表面最大，中心为零。
- 圆轴扭转强度条件：
  - `τmax = T/Wp <= [τ]`
- 极惯性矩 `Ip`。
- 抗扭截面系数 `Wp`。
- 扭转角：
  - `φ = Tl / GIp`
- 刚度条件：
  - `φ <= [φ]`

### 可能会怎么考

- 给传动轴功率和转速，求外力偶矩。
- 画扭矩图。
- 求最大切应力。
- 校核轴的强度或刚度。
- 比较实心轴和空心轴的抗扭能力。

### 易错点

- `Ip` 和弯曲的 `Iz` 混用。
- `T`、`τ`、`φ` 单位混乱。
- 忘了最大切应力在圆轴外表面。
- 空心圆轴公式漏掉内径项。

## 17专题2-弯曲正应力.pdf

### 这个 PPT 大概讲什么

这是弯曲正应力专题，是考试高优先级材料。前面复习梁的类型、弯矩和剪力，后面推导弯曲正应力公式，并讲危险点和截面几何量。

### 必须掌握的知识点

- 梁的三种基本形式：
  - 简支梁。
  - 悬臂梁。
  - 外伸梁。
- 弯曲内力：
  - 剪力 `Fs`。
  - 弯矩 `M`。
- 剪力图和弯矩图的基本步骤：
  1. 求支座反力。
  2. 建立坐标系。
  3. 在载荷变化处分段。
  4. 列每段剪力方程和弯矩方程。
  5. 作图。
- 纯弯曲：横截面只有弯矩，没有剪力。
- 横力弯曲：横截面既有弯矩，又有剪力。
- 中性层、中性轴。
- 平面假设：横截面变形后仍为平面。
- 弯曲正应力分布：
  - 与到中性轴距离成正比。
  - 中性轴一侧受拉，一侧受压。
- 弯曲正应力公式：
  - `σ = My/Iz`
- 最大正应力：
  - `σmax = Mmax/Wz`
- 常见截面危险点：
  - 矩形截面：顶边和底边。
  - 实心圆：圆周外缘上下点。
  - 空心圆：大圆周外缘上下点。

### 可能会怎么考

- 给梁和载荷，求某截面某点的正应力。
- 求指定截面最大正应力。
- 求全梁最大正应力。
- 给截面形状，判断危险点在哪里。
- 判断桥面或梁上/下缘是拉应力还是压应力。
- 问公式 `σ = My/Iz` 的适用条件。

### 易错点

- `y` 是点到中性轴的距离，不是随便取截面高度。
- 最大正应力不一定在题目给的截面，而在 `Mmax` 对应截面。
- 拉压方向和弯矩正负号没统一。
- 混淆纯弯曲和横力弯曲。
- 忘记 `Wz = Iz/ymax`。

## 弯曲内力补充内容：剪力图与弯矩图特征.pdf

### 这个 PDF 大概讲什么

它是一张速查表，非常像选择题、判断题、画图题的核心依据。

### 必须掌握的知识点

载荷、剪力图、弯矩图之间的关系：

- 向下均布载荷：
  - 剪力图：向下倾斜直线。
  - 弯矩图：上凸二次曲线。
  - `Mmax` 可能在 `Fs=0` 的截面。
- 无载荷段：
  - 剪力图：水平直线。
  - 弯矩图：斜直线。
- 集中力处：
  - 剪力图：突变。
  - 弯矩图：转折。
  - `Mmax` 可能在剪力变号的截面。
- 集中力偶处：
  - 剪力图：无变化。
  - 弯矩图：突变。
  - `Mmax` 可能在紧靠集中力偶的某一侧截面。

### 可能会怎么考

- 给载荷图，判断剪力图/弯矩图形状。
- 给剪力图/弯矩图，反推出载荷类型。
- 问集中力和集中力偶分别让哪个图突变。
- 判断最大弯矩可能在哪里。

### 易错点

- 集中力不是让弯矩图突变，而是让弯矩图转折。
- 集中力偶不是让剪力图突变，而是让弯矩图突变。
- `Fs=0` 是弯矩极值的高频位置，但不是唯一位置。

## 弯曲习题.pdf

### 这个 PDF 大概讲什么

这是第八章平面弯曲的习题集，题型覆盖选择、判断、计算。它比课件更能说明考试会怎么问。

### 习题暴露出的重点

- 集中力、集中力偶、均布载荷对内力图的影响。
- 根据剪力图和弯矩图反推载荷。
- 截面法求剪力和弯矩。
- 分段列 `Fs(x)` 和 `M(x)`。
- 画剪力图和弯矩图。
- 用 `σmax = Mmax/Wz` 做强度校核。
- 反求最小截面尺寸。
- 已知许用应力反求许用载荷。
- 用积分法求转角和挠曲线。

### 可能会怎么考

#### 选择题

- 集中力作用处，`Fs` 图突变，`M` 图转折。
- 集中力偶作用处，`M` 图突变，`Fs` 图不变。
- 均布载荷段，`Fs` 图是斜直线，`M` 图是二次曲线。
- 剪力为零处，弯矩可能取极值。
- 同跨度、同载荷、同 `EI` 的梁，内力和挠度是否相同。

#### 计算题

典型步骤：

1. 求支座反力。
2. 按载荷变化处分段。
3. 列 `Fs(x)`。
4. 列 `M(x)`。
5. 画剪力图和弯矩图。
6. 找 `Mmax`。
7. 用 `σmax = Mmax/Wz` 校核或设计截面。

#### 变形题

典型步骤：

1. 写弯矩方程 `M(x)`。
2. 建立 `EIw'' = M(x)`。
3. 积分两次。
4. 用边界条件确定常数。
5. 求指定点转角 `θ` 和挠度 `w`。

### 易错点

- 截面 1 和截面 2 在集中力偶两侧时，弯矩可能不一样。
- 画图时正负号体系前后不一致。
- 固定端最大弯矩容易漏掉。
- `kN·m`、`N·m`、`MPa`、`m^3` 单位换算很容易错。
- 圆截面 `Wz = πd^3/32`，不要写成惯性矩公式。

## 第 19课 弯曲4 提高强度和刚度的措施15.pdf

### 这个 PPT 大概讲什么

它包含三块：弯曲变形积分法、叠加法、提高弯曲强度和刚度的措施。这个文件是弯曲专题的后半段，偏综合计算和工程判断。

### 必须掌握的知识点

#### 1. 用积分法求弯曲变形

核心公式：

- `EIw'' = M(x)`
- `EIw' = ∫M(x)dx + C`
- `EIw = ∬M(x)dxdx + Cx + D`

其中：

- `w` 是挠度。
- `θ = w'` 是转角。
- `C, D` 由边界条件和连续条件确定。

常用边界条件：

- 固定端：`w=0, θ=0`。
- 铰支座/滚动支座：`w=0`。
- 对称点：`θ=0`。
- 分段交界处：挠度和转角连续。

#### 2. 叠加法

多个载荷共同作用时：

- `θ = θ1 + θ2 + ...`
- `w = w1 + w2 + ...`

也就是把复杂载荷拆成简单载荷，再把变形相加。

#### 3. 提高强度和刚度

强度条件：

- 正应力：`σmax = Mmax/Wz <= [σ]`
- 切应力：`τmax = Fs,max Sz*/(bIz) <= [τ]`

刚度条件：

- 挠度：`|wmax|/L <= [w/L]`
- 转角：`|θmax| <= [θ]`

提高弯曲强度和刚度的思路：

- 降低 `Mmax`。
- 提高 `Wz` 和 `Iz`。
- 合理布置载荷和支座。
- 选择合理截面，如工字形、空心截面、变截面梁。
- 根据材料特性选择截面形状。

### 可能会怎么考

- 给简支梁或悬臂梁，求转角方程和挠曲线方程。
- 求 `θmax` 和 `wmax`。
- 给多个载荷，用叠加法求某点挠度。
- 判断哪种支座布置能降低最大弯矩。
- 比较圆形、矩形、空心圆、工字形截面的抗弯能力。
- 问“换高强度材料能不能提高刚度”。
- 给截面面积相同的几种截面，选最合理的抗弯截面。

### 易错点

- `EIw'' = M(x)` 的正负号要和弯矩正负号统一。
- 只用边界条件，忘了分段连续条件。
- 把强度和刚度混为一谈。
- 只记 `σ`，忘记剪应力强度条件也可能考。
- 同类材料强度差别可能大，但 `E` 差别不大，所以提高强度不等于提高刚度。
- 工字形截面提高抗弯效率，是因为材料远离中性轴，提高 `Iz/Wz`。

## 第 20课 压杆稳定.pdf

### 这个 PPT 大概讲什么

这是压杆稳定专题。它讲压杆为什么会失稳、欧拉临界力公式、不同支承条件下的长度系数、柔度判别、稳定校核和提高稳定性的措施。

### 必须掌握的知识点

#### 1. 稳定问题的本质

压杆不是先强度破坏，而可能在轴向压力未达到强度极限前突然弯曲失稳。

#### 2. 两端铰支细长压杆欧拉公式

推导主线：

- 弯矩：`M(x) = -Fw`
- 挠曲线近似方程：`w'' = M/EI`
- 得到：`w'' + k²w = 0`
- 非零解条件导出：

```text
Fcr = π²EI/l²
```

适用条件：

- 理想压杆。
- 在线弹性范围内。
- 两端为球铰支座。

#### 3. 不同约束下的统一公式

```text
Fcr = π²EI/(μl)²
```

长度系数：

- 两端铰支：`μ = 1`
- 一端固定、一端铰支：`μ ≈ 0.7`
- 两端固定：`μ = 0.5`
- 一端固定、一端自由：`μ = 2`

#### 4. 临界应力与柔度

- `σcr = Fcr/A`
- `i = sqrt(I/A)`
- `λ = μl/i`
- 大柔度杆：
  - `λ >= λp`
  - 可用欧拉公式。
- 中柔度杆：
  - `λs <= λ <= λp`
  - 用经验公式，如 `σcr = a - bλ`。
- 小柔度杆：
  - `λ <= λs`
  - 本质是强度问题。

#### 5. 稳定条件

```text
n = Fcr/F >= nst
```

其中 `F` 是工作压力，`nst` 是稳定安全因数。

#### 6. 提高压杆稳定性

- 选择合理截面形状，提高惯性半径 `i`。
- 改变约束条件，减小长度系数 `μ`。
- 减小杆长 `l`。
- 合理选择材料。
- 避免初弯曲、偏心受压、材料不均、支座缺陷。

### 可能会怎么考

- 给压杆约束条件，选 `μ` 并计算 `Fcr`。
- 给矩形截面改变摆放方向，比较临界力变几倍。
- 给复杂结构，先用静力学求某杆轴力，再校核稳定性。
- 给材料参数和杆长截面，判断是大柔度杆还是中柔度杆。
- 用 `n=Fcr/F` 判断是否满足稳定要求。
- 问提高压杆稳定性的措施。

### 易错点

- 直接套欧拉公式，不先判断柔度范围。
- 忘记用最小惯性矩方向计算稳定。
- `l` 和 `μl` 混淆。
- `i=sqrt(I/A)` 算错。
- 把工作压力 `F` 和临界力 `Fcr` 混为一谈。
- 中柔度杆不能用欧拉公式。

## 5. 最像考试的综合题模板

### 模板 1：弯曲内力图题

题目形式：

```text
给一根梁，受集中力、集中力偶、均布载荷作用。
求支座反力，列剪力方程和弯矩方程，画剪力图和弯矩图。
```

答题路线：

1. 画受力图。
2. 求支座反力。
3. 按载荷变化点分段。
4. 对每段列 `Fs(x)` 和 `M(x)`。
5. 标出图上关键值。
6. 检查集中力处 `Fs` 是否突变，集中力偶处 `M` 是否突变。

### 模板 2：弯曲强度校核题

题目形式：

```text
给梁、载荷、截面尺寸和许用应力，校核梁是否安全，或反求最小尺寸。
```

答题路线：

1. 先画弯矩图。
2. 找 `Mmax`。
3. 求截面系数 `Wz`。
4. 用 `σmax = Mmax/Wz <= [σ]`。
5. 如果是设计题，反解截面尺寸。

### 模板 3：弯曲变形题

题目形式：

```text
给梁和载荷，已知 EI，求转角方程、挠曲线方程、某点转角或挠度。
```

答题路线：

1. 建坐标。
2. 写 `M(x)`。
3. 写 `EIw'' = M(x)`。
4. 积分两次。
5. 用边界条件和连续条件定常数。
6. 代入指定位置求 `θ` 和 `w`。

### 模板 4：压杆稳定题

题目形式：

```text
给压杆长度、截面、材料、约束条件和工作压力，校核稳定性。
```

答题路线：

1. 确定约束条件，选 `μ`。
2. 计算截面面积 `A`、惯性矩 `I`。
3. 取最小惯性矩方向。
4. 计算 `i=sqrt(I/A)`。
5. 计算 `λ=μl/i`。
6. 判断大柔度、中柔度、小柔度。
7. 选择欧拉公式或经验公式。
8. 算 `Fcr`。
9. 用 `n=Fcr/F >= nst` 校核。

## 7. 最容易丢分的地方

- 支座反力方向画错。
- 固定端漏掉固定端弯矩。
- 分段点漏掉集中力或集中力偶。
- 集中力和集中力偶对图形的影响记反。
- 弯矩图不标关键值。
- `Mmax` 找错，导致后面强度校核全错。
- 截面系数 `Wz` 和惯性矩 `Iz` 混用。
- 单位没有统一到 `N, m, Pa`。
- 弯曲变形题只会积分，不会用边界条件。
- 压杆稳定不先判断柔度。
- 压杆没有取最小惯性矩方向。
- 把强度问题、刚度问题、稳定问题混在一起。
