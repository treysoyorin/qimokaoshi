## Course Overview

化工原理是化工、生物、冶金、食品等专业的重要技术基础课，研究化工生产中各种物理操作的基本原理、典型设备结构、操作性能和设计计算。核心方法论：质量衡算、能量衡算、动量衡算。

课程特点（老师原话）：**"符号多、公式多、单位换算多"**。研究真实、复杂的生产问题：特定物料、特定设备、特定过程。

## 绪论 核心内容

### 1. 单元操作（Unit Operations）
- **定义**：化工生产过程中共有的物理操作，只改变物料状态或物理性质，不改变化学性质。
- **分类**：
  - 流体动力过程：流体输送、沉降、过滤、固体流态化
  - 热过程：传热、冷凝、蒸发
  - 传质过程：蒸馏、吸收、萃取、结晶、干燥
- **特点**：不同化工生产中包含的单元操作数目、名称与排列顺序不同，但基本原理和设备通用。

### 2. 三种衡算（基本分析方法）
- **质量衡算**：输入 = 输出 + 积累（稳态时积累=0）
- **能量衡算**：输入能量 = 输出能量 + 损失
- **动量衡算**：过程速率 = 推动力 / 阻力

### 3. 研究方法
- 理论模型、经验模型、**半经验半理论模型**（重点！）、人工智能模型
- 量纲分析（π定理）：无量纲数群个数 = 变量数 - 基本量纲个数

### 4. 单位换算（必须非常熟悉）
- 1 atm = 1.033 kgf/cm² = 760 mmHg = 10.33 mH₂O = 1.0133 bar = 101325 Pa
- 1 kgf/cm² = 1 at = 9.80×10⁴ Pa = 735.6 mmHg = 10 mH₂O
- 1 kgf = 9.81 N

### 5. 重点例题（绪论）
- **例0-1**：硝酸钾生产物料衡算（带循环，蒸发+结晶）
- **例0-2**：换热器热量衡算（求热损失百分数）

## 第一章 流体流动原理及应用（14学时）

### 1.1 流体基本概念（重点：密度计算）
- **理想气体密度**：ρ = PM/(RT) 或 ρ = ρ₀·(P/P₀)·(T₀/T)
- **理想气体混合物**：ρ_m = Σ(y_i·M_i)·P/(RT) 或 1/ρ_m = Σ(w_i/ρ_i)
- **液体混合物**：1/ρ_m = Σ(w_i/ρ_i)（质量分数）
- **例1-1**：HCl混合气体（25% HCl、75%空气）密度计算
- **例1-2**：60%醋酸水溶液密度计算

### 1.2 流体静力学（★★★重点章节）
- **压强表示**：绝对压强、表压、真空度
  - 表压 = 绝对压强 - 大气压
  - 真空度 = 大气压 - 绝对压强
- **静力学基本方程**：P₂ = P₁ + ρgh（适用条件：静止、连续、同一种流体、密度为常数）
- **等压面**：静止、连续、均一、水平。正确选取等压面是解题关键！
- **U形管压差计**：ΔP = (ρ_A - ρ_B)·g·R + ρ_B·g·Z（当被测管水平时Z=0，简化为ΔP=(ρ_A-ρ_B)gR）
- **例1-3**：天津vs兰州真空度计算（海拔对大气压的修正！）
- **例1-4**：开口容器油水界面等压面判断（★经典题）
- **例1-5**：复式U形管压差计（A、B两点间压差，R₁=0.35m，R₂=0.45m）
- **例1-6**：鼓泡式液面测量（液面高度h计算）
- **例1-7**：液封高度计算（真空度86kPa时气压管水上升高度）

### 1.3 流体流动基本概念（重点：雷诺数、流动型态）
- **连续性方程**：u₁A₁ = u₂A₂，不可压缩时 u₂/u₁ = (d₁/d₂)²
- **牛顿黏性定律**：τ = μ·(du/dy)
- **动力粘度单位**：Pa·s，1 cP = 10⁻³ Pa·s = 1 mPa·s
- **流动型态**：
  - Re ≤ 2000：层流
  - Re ≥ 4000：湍流
  - 2000 < Re < 4000：过渡区（不稳定）
- **速度分布**：
  - 层流：抛物线，u = u_max·[1-(r/R)²]，平均速度 u = u_max/2
  - 湍流：较平坦，u ≈ 0.82u_max（n=7时）
- **边界层**：层流内层是传递过程的主要阻力，Re越大层流内层越薄

### 1.4 伯努利方程（★★★★★核心中的核心）
- **机械能衡算式**：
  ```
  gz₁ + u₁²/2 + P₁/ρ + W_e = gz₂ + u₂²/2 + P₂/ρ + Σh_f
  ```
- **压头形式**：z₁ + u₁²/(2g) + P₁/(ρg) + H_e = z₂ + u₂²/(2g) + P₂/(ρg) + H_f
- **适用条件**：不可压缩、稳态、连续
- **理想流体**：Σh_f = 0，任一截面总机械能不变
- **关键讨论**：
  - 各项单位必须统一
  - 基准面选取后不能换
  - 动压头不要漏
  - 泵的扬程与管路所需压头关系

### 1.5 流体流动阻力计算（★★★★★期末大题核心）
- **范宁公式**：
  ```
  h_f = λ·(l/d)·(u²/2g)     [压头损失，m]
  ΔP_f = λ·(l/d)·(ρu²/2)    [压强损失，Pa]
  ```
- **层流**：λ = 64/Re（精确，哈根-泊谡叶公式）
- **湍流**：
  - 光滑管：Blasius公式 λ = 0.3164/Re^0.25（Re=3000~10⁵）
  - 顾毓珍公式（我国科学家贡献！）
  - 粗糙管：Nikurades-Karman公式
  - **Moody图**：根据Re和ε/d查λ（★考试中常用）
- **摩擦因数图分区**：
  - 层流区：Re≤2000，λ=64/Re
  - 过渡区：2000<Re<4000，波动
  - 湍流区：Re≥4000，λ随Re减小、随ε/d增大
  - 完全湍流区（阻力平方区）：λ≈常数，h_f∝u²
- **局部阻力**：
  - 阻力系数法：h_f = ζ·u²/(2g)
  - 当量长度法：h_f = λ·(l_e/d)·u²/(2g)
  - 常见ζ：突然扩大ζ=(1-A₁/A₂)²，进口ζ=0.5，出口ζ=1，90°弯头ζ=0.75，截止阀全开ζ=6.0
- **非圆形管**：当量直径 de = 4×流通截面积/润湿周边长
  - 矩形管（a×b）：de = 2ab/(a+b)
  - 套管环隙：de = d₂ - d₁
- **例1-14**：甘油层流（20°C，μ=1499mPa·s，层流，每米管长压降）
- **例1-15**：湍流阻力计算（u=4.5m/s，d=0.05m，查Moody图得λ=0.024）
- **例1-16**：管径选择（根据流量和流速初选管径，再圆整、核算）
- **例1-17**：试差法计算管径（★经典迭代题，设u→算d→圆整→算Re→查λ→验算压头）

### 1.6 管路计算
- **简单管路**：流量不变，总阻力=各段之和
- **复杂管路**：
  - 并联管路：各支管压降相等，总流量=各支管之和
  - 分支管路：分支点总压头恒定
- **试差法**：当Re无法确定（因d或u未知）时，先假设λ（如取阻力平方区值），迭代至收敛

### 1.7 流体输送机械（★★★重点，泵）
- **离心泵工作原理**：灌泵→排液→吸液，依靠离心力
- **气缚现象**：泵内充满气体无法吸液，启动前必须灌泵
- **主要性能参数**：
  - 流量Q（m³/s）
  - 扬程H（m液柱）——泵对单位质量流体提供的有效压头
  - 效率η（50%~90%）
  - 轴功率N = ρgQH/(1000η)（kW）
- **特性曲线**：H-Q、N-Q、η-Q
  - 启动时关闭出口阀（N最小，保护电机）
  - 最高效率点对应最佳工况参数
- **比例定律**：Q∝n，H∝n²，N∝n³（n变化<20%）
- **汽蚀现象**：
  - 产生条件：p_k ≤ p_v（饱和蒸气压）
  - 后果：震动、噪音、腐蚀、流量压头效率下降
  - 避免条件：p_k > p_v
- **安装高度**：
  ```
  H_g = P_a/(ρg) - P_v/(ρg) - NPSH_r - H_f,吸入
  ```
  - 敞开贮槽：H_g = (P_a - P_v)/(ρg) - NPSH_r - H_f
  - 安全裕量：0.5m
- **例1-18**：IS65-40-200泵安装高度计算（50°C水，NPSH_r=2.0m，当地大气压100kPa，吸入阻力2m）
- **工作点**：泵特性曲线与管路特性曲线交点
- **流量调节**：
  - 调出口阀：改变管路特性，简捷但不经济
  - 改变转速：改变泵特性，经济但需要变速装置
- **泵的组合**：
  - 低阻管路：并联优于串联
  - 高阻管路：串联优于并联
- **泵的选型**：根据液体性质→确定Q和H→查样本选型号→核算轴功率
- **泵的启动/关闭**：
  - 启动：灌泵→关出口阀→启动电机→逐渐开阀
  - 关闭：关出口阀→停电机→关入口阀

## 第二章 传热及传热设备（14学时）

### 2.1 传热基本概念
- **三种传热方式**：热传导（微观粒子热运动）、对流传热（流体宏观运动+导热）、热辐射（电磁波，不需介质）
- **化工生产中的传热要求**：强化传热（换热设备）vs 削弱传热（保温）

### 2.2 热传导（★★★重点）
- **傅里叶定律**：dQ = -λ·S·(dt/dx)，λ为热导率（W/(m·K)）
- **单层平壁**：Q = λSΔt/b = Δt/R，温度分布为直线
- **多层平壁**：Q = Δt/Σ(bᵢ/λᵢS)，串联热阻，温差分配∝热阻
- **单层圆筒壁**：Q = 2πλLΔt/ln(r₂/r₁)，温度分布为对数曲线，q随r变化
- **多层圆筒壁**：Q = Δt/Σ[ln(rᵢ₊₁/rᵢ)/(2πλᵢL)]，各层用各自对数平均面积
- **接触热阻**：实际工程中两层接触处因粗糙产生额外热阻
- **例2-1**：单层平壁求q
- **例2-2**：多层平壁炉墙（耐火砖+硅藻土+金属板），求散热速率和界面温度
- **例2-3**：多层圆筒壁管路保温，求每米管长热损失

### 2.3 对流传热（★★★重点）
- **牛顿冷却定律**：Q = αSΔt，α为对流传热系数（W/(m²·K)）
- **对流传热机理**：层流内层是主要热阻（虽薄但λ小）
- **影响α的因素**：流体种类、相变化、物性、流动状况、流动原因、传热面形状
- **特征数关联式**：Nu = f(Re, Pr, Gr)
- **管内湍流强制对流（Dittus-Boelter）**：Nu = 0.023·Re^0.8·Pr^n
  - 液体加热n=0.4，冷却n=0.3；气体同样取0.4/0.3
  - 适用范围：Re>10⁴，0.7<Pr<120，L/di>60
  - 定性温度：流体平均温度；特性尺寸：管内径di
- **高黏度流体（Sieder-Tate）**：Nu = 0.027·Re^0.8·Pr^(1/3)·(μ/μw)^0.14
- **层流（Sieder-Tate）**：Nu = 1.86·(Re·Pr·di/L)^(1/3)·(μ/μw)^0.14
- **管外强制对流**：管束外垂直流动（直列/错列不同公式）、管壳式壳程（Donohue/Kern法）
- **有相变时α大得多**：
  - 蒸汽冷凝：膜状冷凝（液膜热阻）vs 滴状冷凝（α大5~10倍）；不凝气体使α↓↓
  - 液体沸腾：核状沸腾（α大、热通量高）vs 膜状沸腾（α骤降）；工程控制在核状沸腾区
- **例2-4/例2-5**：对流传热系数α计算（P85）
- **例2-6**：对流辐射联合传热系数（P94）

### 2.4 热辐射
- **热辐射特点**：不需介质，0.4~40μm，高温→低温传递
- **吸收/反射/透过**：A+R+D=1；固体D≈0，气体R≈0
- **黑体/白体/灰体**：A=1/R=1/ε为常数；大多数工程材料视为灰体
- **斯蒂芬-玻尔兹曼定律**：E₀ = C₀(T/100)⁴，C₀=5.67 W/(m²·K⁴)，T必须用K！
- **灰体辐射**：E = εC₀(T/100)⁴，克希霍夫定律A=ε
- **两灰体间辐射传热**：Q = C₁₋₂φS[(T₁/100)⁴-(T₂/100)⁴]
- **对流辐射联合传热**：Q = αT·S·(tw-t)，αT为联合传热系数

### 2.5 间壁式换热器传热计算（★★★★★核心）
- **传热过程**：热流体对流→壁面导热→冷流体对流（三步骤串联）
- **热量衡算**：
  - 无相变：Q = Wh·Cph·(T₁-T₂) = Wc·Cpc·(t₂-t₁)
  - 有相变：Q = Wh·r = Wc·Cpc·(t₂-t₁)
- **总传热速率方程**：Q = KSΔtm
- **总传热系数K**：
  - 1/K₀ = 1/α₀ + bd₀/(λdm) + d₀/(αᵢdᵢ) + Rso + Rsi(d₀/dᵢ)（管外基准）
  - dm = (d₀-dᵢ)/ln(d₀/dᵢ) = 对数平均直径
  - **K由α小侧控制**：K≈min(α₀, αᵢ)，提高K需提高α较小一侧
  - **污垢热阻不可忽视**：通常比传热壁热阻大得多
- **平均温度差Δtm**：
  - 恒温传热：Δtm = T-t
  - 逆流/并流：Δtm = (Δt₁-Δt₂)/ln(Δt₁/Δt₂)，当Δt₁/Δt₂<2可用算术平均
  - 逆流Δtm > 并流Δtm，工业上多用逆流
  - 错流/折流：Δtm = Δtm,逆流 × φΔt，φΔt查图，一般>0.8
- **换热器计算类型**：
  - 设计型：根据热负荷求S
  - 操作型：对现有换热器判断适用性（常需试差迭代）
- **强化传热**：增大Δtm、增大S、减小热阻（提高α小侧、减薄壁厚、除垢）
- **ε-NTU法**：
  - ε = Q/Qmax = 实际/最大可能传热量
  - NTU = KS/(WCp)min
  - 查图得ε，再求出口温度
  - 优点：出口温度未知时无须试差
- **例2-7**：K值计算（管壳式换热器，含污垢热阻，K由α小侧控制）
- **例2-8**：并流/逆流平均温度差计算（比较Δtm）
- **例2-9**：设计型计算（套管换热器，冷却苯，求冷却水量和总管长）
- **例2-10**：操作型计算（水量加倍后Q'/Q，需试差）
- **例2-11**：ε-NTU法（求逆流换热器出口温度和传热量）

### 2.6 换热设备
- **换热器类型**：直接接触式、蓄热式、间壁式（管式/板式/翅片式）
- **管壳式换热器**：固定管板式、浮头式、U形管式（各有优缺点）
- **折流挡板**：引导横向流动、增强传热、支撑管束
- **强化传热途径**：增大Δtm、增大S、增大K（提高α小侧、除垢）
- **管壳式换热器设计步骤**：估算面积→选类型→算温度差→选K→初算S→确定尺寸→算压降→校核K计/K选=1.15~1.25
- **流体通道选择**：不清洁/易结垢/高压→管程；饱和蒸汽/被冷却/α大→壳程
- **例2-12**：管壳式换热器设计举例

## Common Traps（高频扣分点）

1. **压强项单位不统一**：Pa vs mmHg vs mH₂O，必须换算统一
2. **等压面选取错误**：B-B'不是等压面（不连通或不同流体），A-A'是等压面（连通、同种、静止、水平）
3. **伯努利方程中基准面乱换**：z是相对高度，选定后不能改
4. **d用内径**：φ57×3.5mm钢管，内径=57-2×3.5=50mm
5. **Moody图读错**：Re和ε/d两个参数都要对，过渡区结果波动
6. **局部阻力系数用错流速**：突然扩大用进口（小管）流速，突然缩小用出口（小管）流速
7. **泵安装高度不修正**：温度高时P_v很大，不能忽略
8. **NPSH_a vs NPSH_r搞混**：a=available（实际提供），r=required（泵要求）
9. **功率单位**：Q用m³/s（不是m³/h），结果单位kW
10. **真空度计算**：兰州大气压85.3kPa vs 天津101.33kPa，真空度不同
11. **混合液体密度**：用质量分数加权，1/ρ_m = Σ(w_i/ρ_i)，不是算术平均
12. **层流内层概念**：湍流中靠近壁面仍有层流内层，是传递过程的主要阻力
