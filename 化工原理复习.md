## 30 秒总结

化工原理 = **单元操作 + 三种衡算 + 工程观念**。期末核心在**第一章（流体流动，14学时）**和**第二章（传热，14学时）**，各占约1/3学时。

先抓第1章：
1. **伯努利方程**（流量、压差、功、位差计算）
2. **阻力计算**（范宁公式 + Moody图 + 局部阻力）
3. **泵安装高度**（汽蚀余量 + 饱和蒸气压修正）

再抓第2章：
4. **总传热速率方程**（Q=KSΔtm，热量衡算+传热速率方程联立）
5. **K值计算**（1/K₀ = 1/α₀ + bd₀/(λdm) + d₀/(αᵢdᵢ) + 污垢热阻，K由α小侧控制）
6. **对数平均温差**（Δtm = (Δt₁-Δt₂)/ln(Δt₁/Δt₂)，逆流>并流）

## Must Know（每章核心公式）

### 绪论
- 三种衡算：质量衡算、能量衡算、动量衡算
- 单位换算：1 atm = 101325 Pa = 760 mmHg = 10.33 mH₂O
- 理想气体密度：ρ = PM/(RT)（公式3）
- 混合液体密度：1/ρ_m = Σ(w_i/ρ_i)（公式8）

### 流体静力学
- **静力学方程**：P₂ = P₁ + ρgh（公式10）
- **U形管压差计**：ΔP = (ρ_A - ρ_B)gR（水平管时）
- **等压面条件**：静止、连续、同种、水平
- **表压/真空度**：表压 = 绝压 - 大气压；真空度 = 大气压 - 绝压

### 流体动力学
- **连续性方程**：u₁A₁ = u₂A₂，不可压缩时 u₂/u₁ = (d₁/d₂)²
- **雷诺数**：Re = ρud/μ，Re≤2000层流，Re≥4000湍流
- **牛顿黏性定律**：τ = μ(du/dy)
- **粘度单位**：1 cP = 10⁻³ Pa·s = 1 mPa·s

### 伯努利方程（★★★）
```
gz₁ + u₁²/2 + P₁/ρ + W_e = gz₂ + u₂²/2 + P₂/ρ + Σh_f
```
- 单位：J/kg（能量衡算式）
- 压头形式：z + u²/(2g) + P/(ρg) + H_e = z + u²/(2g) + P/(ρg) + H_f
- 单位：m（每一项都是"液柱高度"）
- **适用条件**：不可压缩、稳态、连续
- **理想流体**：Σh_f = 0，总机械能守恒

### 阻力计算（★★★）
- **范宁公式**：
  ```
  h_f = λ(l/d)(u²/2g)        [压头损失，m]
  ΔP_f = λ(l/d)(ρu²/2)       [压强损失，Pa]
  ```
- **层流**：λ = 64/Re（精确）
- **湍流光滑管**：Blasius λ = 0.3164/Re^0.25（Re=3000~10⁵）
- **湍流粗糙管**：查 **Moody图**（图1-19）
- **摩擦因数图分区**：
  - 层流区：Re≤2000，λ=64/Re
  - 过渡区：2000<Re<4000
  - 湍流区：Re≥4000，λ随Re减小、随ε/d增大
  - 完全湍流区（阻力平方区）：λ≈常数，h_f∝u²
- **局部阻力**：
  - 阻力系数法：h_f = ζ·u²/(2g)
  - 当量长度法：h_f = λ(l_e/d)(u²/2g)
  - 常见ζ：进口0.5，出口1，90°弯头0.75，截止阀全开6.0
- **非圆形管**：de = 4A/Π，矩形de=2ab/(a+b)

### 传热（★★★新增）

- **傅里叶定律**：Q = λSΔt/b = Δt/R（平壁热传导）
- **多层平壁**：Q = Δt/Σ(bᵢ/λᵢS)，温差分配∝热阻
- **圆筒壁**：Q = 2πλLΔt/ln(r₂/r₁)，温度分布对数曲线
- **牛顿冷却定律**：Q = αSΔt（对流传热）
- **管内湍流强制对流**：Nu = 0.023·Re^0.8·Pr^n（加热n=0.4，冷却n=0.3）
- **总传热速率方程**：Q = KSΔtm（换热器核心）
- **总传热系数**：1/K₀ = 1/α₀ + bd₀/(λdm) + d₀/(αᵢdᵢ) + Rso + Rsi(d₀/dᵢ)
- **K值控制**：K≈α小侧（热阻大侧控制），提高α小侧才能有效提高K
- **对数平均温差**：Δtm = (Δt₁-Δt₂)/ln(Δt₁/Δt₂)，逆流>并流
- **黑体辐射**：E₀ = C₀(T/100)⁴（绝对温度！四次方关系）
- **灰体辐射**：E = εC₀(T/100)⁴，A=ε（克希霍夫定律）
- **辐射传热**：Q = C₁₋₂φS[(T₁/100)⁴-(T₂/100)⁴]
- **ε-NTU法**：ε = Q/Qmax，NTU = KS/(WCp)min，查图求出口温度

### 泵（★★★）
- **轴功率**：N = ρgQH/(1000η)（kW，Q用m³/s）
- **比例定律**：Q∝n，H∝n²，N∝n³（n变化<20%）
- **安装高度**（★必考）：
  ```
  H_g = P_a/(ρg) - P_v/(ρg) - NPSH_r - H_f,吸入
  ```
  - 敞开贮槽：H_g = (P_a - P_v)/(ρg) - NPSH_r - H_f
  - 安全裕量：0.5m
- **汽蚀条件**：p_k > p_v（避免汽蚀）；p_k ≤ p_v（发生汽蚀）
- **NPSH**：泵入口处液体的静压头+动压头 - 饱和蒸气压头的最小允许值
- **工作点**：泵H-Q曲线与管路H_e-Q_e曲线的交点
- **流量调节**：调出口阀（简捷但不经济）vs 改变转速（经济但需变速装置）

## 老师可能会这么问（基于课件例题）

### 计算题类型
1. **"计算混合气体/混合液体密度"**
   - 对应：例1-1（HCl混合气体）、例1-2（醋酸水溶液）
   - 关键：气体用摩尔分数，液体用质量分数；注意状态条件

2. **"判断等压面是否成立，并计算某点压强/液柱高度"**
   - 对应：例1-4（油水界面等压面判断）、例1-5（复式U形管）
   - 关键："同一水平面上连通流体的静压强相等"；不连通或不同流体则不等

3. **"用伯努利方程计算管路某截面压强或泵所需功率"**
   - 对应：伯努利方程推导后的应用
   - 关键：选好基准面和截面，检查单位，动压头不要漏

4. **"判断流动型态，计算直管阻力损失"**
   - 对应：例1-14（甘油层流）、例1-15（湍流查Moody图）
   - 关键：先算Re→判断层流/湍流→选公式/查图→代入范宁公式

5. **"选择管径（含试差法）"**
   - 对应：例1-16（初选流速+圆整+核算）、例1-17（试差法迭代）
   - 关键：试差法先设u→算d→圆整→算Re→查λ→验算压头

6. **"计算离心泵安装高度，判断是否汽蚀"**
   - 对应：例1-18（IS65-40-200泵，50°C水）
   - 关键：查饱和蒸气压、修正大气压、安全裕量0.5m

7. **"泵的工作点与流量调节分析"**
   - 对应：阀门调节、转速调节的图解分析
   - 关键：调阀→改变管路特性；调转速→改变泵特性

8. **"泵的选型与组合方式选择"**
   - 对应：例1-20、泵串联/并联
   - 关键：低阻管路并联优于串联；高阻管路串联优于并联

9. **"计算平壁/圆筒壁热传导速率或界面温度"**
   - 对应：例2-1（单层平壁）、例2-2（多层平壁）、例2-3（多层圆筒壁）
   - 关键：多层壁串联热阻，温差分配∝热阻；圆筒壁用对数平均

10. **"计算总传热系数K，分析如何提高K"**
    - 对应：例2-7（管壳式换热器，含污垢热阻）
    - 关键：K由α小侧控制，1/K₀ = 1/α₀ + bd₀/(λdm) + d₀/(αᵢdᵢ) + Rso + Rsi(d₀/dᵢ)

11. **"计算并流/逆流平均温度差，比较优劣"**
    - 对应：例2-8（热水加热料液）
    - 关键：Δtm = (Δt₁-Δt₂)/ln(Δt₁/Δt₂)；逆流Δtm > 并流Δtm

12. **"换热器设计型计算（求面积或管长）"**
    - 对应：例2-9（套管换热器，冷却苯）
    - 关键：热量衡算Q = WhCphΔT = WcCpcΔt → 求Δtm → 求K → 求S → 求L

13. **"换热器操作型计算（参数变化后求新工况）"**
    - 对应：例2-10（水量加倍后Q'/Q）
    - 关键：联立热量衡算和传热速率方程，常需试差迭代

14. **"用ε-NTU法求换热器出口温度"**
    - 对应：例2-11（逆流换热器，冷热流体出口温度）
    - 关键：找(WCp)min，算NTU = KS/(WCp)min，查图得ε，再求Q和出口温度

15. **"计算两灰体间的辐射传热量"**
    - 对应：辐射传热公式
    - 关键：必须用绝对温度K，四次方关系，C₁₋₂和φ的确定

### 简答题类型
1. **"什么是单元操作？有何特点？"**
   - 答：物理操作、共有操作、基本原理和设备通用

2. **"层流与湍流的本质区别是什么？"**
   - 答：质点运动方式不同。层流分层流动，湍流有随机脉动和位置交换

3. **"伯努利方程的适用条件是什么？"**
   - 答：不可压缩、稳态、连续流体

4. **"什么是汽蚀现象？如何避免？"**
   - 答：p_k ≤ p_v时液体汽化→气泡破灭→冲击叶轮。避免：提高p_k（降低安装高度、减小吸入阻力）或降低p_v（降低液体温度）

5. **"边界层分离的必要条件是什么？"**
   - 答：流体具有黏性 + 流动过程中存在逆压梯度

6. **"三种衡算是什么？在化工原理中的作用？"**
   - 答：质量衡算、能量衡算、动量衡算。是分析问题的基本方法

7. **"三种传热方式的特点和区别？"**
   - 答：热传导（微观粒子热运动，需接触）、对流传热（流体宏观运动+导热）、热辐射（电磁波传播，不需介质）

8. **"为什么K值总是接近α较小的一侧？如何提高K值？"**
   - 答：K由热阻大侧控制。1/K≈1/α小侧。提高K需提高α较小一侧的流速或改变传热面形状

9. **"为什么工业上换热器多采用逆流操作？"**
   - 答：相同进出口温度下，逆流Δtm最大，可节省传热面积或减少介质流量

10. **"什么是污垢热阻？为什么设计中不可忽视？"**
    - 答：传热表面污垢积存导致的热阻。通常比传热壁热阻大得多，会显著降低K值

11. **"蒸汽冷凝中为什么不凝性气体危害很大？"**
    - 答：不凝气体在壁面聚集形成气膜，热阻大增。含1%空气可使α下降60%以上

12. **"液体沸腾为什么要控制在核状沸腾区？"**
    - 答：核状沸腾α大、热通量高；过渡至膜状沸腾后α骤降，壁温可能飞升

## 高频扣分点

1. **单位不统一**：Pa vs mmHg vs mH₂O，必须全部统一为同一种
2. **等压面选取错误**：B-B'不连通不是等压面；不同流体交界处不是等压面
3. **d用内径**：φ57×3.5mm → 内径50mm，不是57mm
4. **Moody图查错**：Re和ε/d两个参数都要对准，别只看一个
5. **局部阻力用错流速**：突然扩大用进口（小管）流速，突然缩小用出口（小管）流速
6. **泵安装高度不修正P_v**：50°C水P_v=12.31kPa，不是0！温度越高P_v越大，安装高度越低
7. **NPSH_a vs NPSH_r搞混**：a=available（实际提供），r=required（泵要求）
8. **功率公式中Q单位**：Q必须用m³/s（50m³/h = 50/3600 m³/s）
9. **混合密度算法**：液体混合物是1/ρ_m = Σ(w_i/ρ_i)，不是算术平均ρ_m = Σ(w_i·ρ_i)
10. **真空度与海拔**：兰州大气压85.3kPa，天津101.33kPa，同一设备真空度不同
11. **伯努利方程中压强项**：P/ρg的单位是m，ρ必须对应液体密度
12. **层流内层概念**：湍流中靠近壁面仍有层流内层，是传递过程的主要阻力，不是"湍流就没有层流"
13. **传热温度用绝对温度**：辐射传热中T必须用K（开尔文），不能用℃
14. **K的基准面积**：工程上以管外面积为基准，注意公式中d₀/dᵢ的换算
15. **污垢热阻不可忽略**：实际K比理论值小很多，设计时必须查经验值
16. **对数平均温差**：Δt₁和Δt₂必须是换热器两端的温差，不是进出口温度本身
17. **多层平壁温度差分配**：热阻大的层温差大，不要想当然平均分配
18. **α关联式定性温度**：多数取流体平均温度，壁温取μw，不要搞混
19. **热流方向n值**：液体加热n=0.4，冷却n=0.3；气体虽然趋势相反但仍取0.4/0.3
20. **平壁vs圆筒壁温度分布**：平壁是直线，圆筒壁是对数曲线，不要混淆
21. **ε-NTU法(WCp)min**：是热容流率（流量×比热），不是单纯流量
22. **逆流并流温差取法**：并流时Δt₁=T₁-t₁，Δt₂=T₂-t₂；逆流时Δt₁=T₂-t₁（或T₁-t₂），注意交叉
23. **操作型计算需试差**：联立热量衡算和传热速率方程，设出口温度→验证→迭代
24. **K计/K选范围**：管壳式换热器设计时，计算K与经验K比值应在1.15~1.25
